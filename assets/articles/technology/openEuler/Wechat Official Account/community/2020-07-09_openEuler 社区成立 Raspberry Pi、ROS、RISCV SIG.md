# [openEuler 社区成立 Raspberry Pi、ROS、RISCV SIG](https://mp.weixin.qq.com/s/byPt_zqfe-4aZ4x80MsSXA)

*openEuler 团队*[OpenAtom openEuler](javascript:void%280%29;)*2020-07-09 19:33:09*

自 2019年12月31日 openEuler 社区上线开放以来，中科院软件所作为社区首批成员，积极参与 openEuler 社区建设。先后在社区主导成立了 RaspberryPi、ROS、RISC-V 等三个 SIG 组，共同推进 openEuler 生态的繁荣发展。目前三个 SIG 的主要 Maintainer 均来自中科院软件所。

建立 Raspberry Pi SIG，主要目标是将 openEuler 操作系统移植到树莓派，以降低 openEuler 操作系统使用门槛，促进其使用与推广。目前 Raspberry Pi SIG 已经完成对 openEuler 操作系统的移植工作，并提供 openEuler 的 Raspberry  Pi 版本镜像的构建脚本和使用文档。

后续会分析整理树莓派相关内核代码，逐步添加到 openEuler 内核主分支，完善 openEuler 树莓派的使用文档、构建脚本等，并根据 openEuler 迭代版本，持续完成 openEuler 向树莓派的移植以及后续维护、升级。

ROS(Robot Operatingsystem) 是一个机器人编程框架,它为机器人各个组件之间的通信搭建了桥梁；ROS 也是一个中间件，连接了操作系统和开发者的 ROS 程序。ROS 分布式的设计理念，支持模块化的定制和修改，使得机器人的感知、决策、控制算法可以更好的组织和运行。

ROS SIG 致力于完成 openEuler 操作系统对机器人分布式通信的底层支持。后续会将 ROS 的官方代码包进行移植和适配，完成后社区贡献者和用户可以直接从社区获取最新的 ROS 包进行安装和使用，同时，ROS SIG 也会提供后续的技术维护。

RISC-V 是一个通用处理器指令集架构（ISA），具有开源、开放、先进、生态协作等技术优势。

RISC-V SIG旨在为 openEuler 构建面向 RISC-V 架构操作系统的研发工作（openEuler4RISC-V）。该工作致力于为推动丰富 RISC-V 的软件生态做出贡献；满足广大技术爱好者、企业、组织等尝试在 RISC-V 环境上开发、使用 openEuler 操作系统的迫切需要；为需要为 RISC-V 硬件定制开发 openEuler 操作系统的技术爱好者提供软件包编译、系统构建、系统定制等技术支持；同时兴趣组也提供了自动化编译、构建工具、构建手册、RPM 托管等资源，使对 RISC-V 感兴趣的开发者能够快速参与到开源系统开发活动中来。

**关于 openEuler**

openEuler 是一个开源、免费的 Linux 发行版平台，将通过开放的社区形式与全球的开发者共同构建一个开放、多元和架构包容的软件生态体系。同时，openEuler 也是一个创新的平台，鼓励任何人在该平台上提出新想法、开拓新思路、实践新方案。openEuler 通过社区合作，打造创新平台，构建支持多处理器架构、统一和开放的操作系统，推动软硬件应用生态繁荣发展。

**关于中科院软件所**

中科院软件所是一所专门从事计算机科学理论研究和软件高技术发展的综合性研究所,中国早期操作系统、高级程序设计语言等软件科学和技术里程碑式的发展，都出自软件所和软件所的前身; 建所三十多年来，软件所对计算机科学与软件的发展做了大量基础性、战略性和前瞻性创新贡献。
