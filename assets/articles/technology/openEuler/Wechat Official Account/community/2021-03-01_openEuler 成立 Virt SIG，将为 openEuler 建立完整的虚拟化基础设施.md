# [openEuler 成立 Virt SIG，将为 openEuler 建立完整的虚拟化基础设施](https://mp.weixin.qq.com/s/Mrq9w80YBTNCU22ESLcsdA)

原创*周荔人*[OpenAtom openEuler](javascript:void%280%29;)*2021-03-01 22:25:24*

经 openEuler 社区技术委员会讨论决定，openEuler 社区正式成立 Virt SIG。Virt SIG 将为 openEuler 建立完整的虚拟化基础设施。并且围绕 StratoVirt 下一代虚拟化技术进行技术实践和探索。openEuler 社区中的开发者可以使用 Virt SIG 提供的虚拟化基础设施，并且在该虚拟化基础设施的基础上开发虚拟化应用。

 

虚拟化作为充分发挥计算机硬件性能的技术，对于操作系统来说是非常重要的。虚拟化技术是国内外芯片和操作系厂商都十分重视的一种技术。openEuler 作为一个支持多样性计算的操作系统，其虚拟化能力要适配多种芯片架构，在Virt SIG 的努力下，openEuler 的虚拟化能力已经在鲲鹏、Arm、x86 等芯片架构上完成适配与优化。

 

根据 Virt SIG 最初制定的技术路标，在 openEuler 20.03 LTS 版本中已经完成了中断虚拟化优化、内存虚拟化优化、存储虚拟化优化路标。提升了 openEuler 在多种硬件设备上的虚拟化能力。在 openEuler 20.09 中，Virt SIG 把 QEMU+KVM、StratoVirt 作为主要的技术研发重点，在该版本发布时，QEMU+KVM、StratoVirt 的以下特性也一并发布：

 

![](https://mmbiz.qpic.cn/mmbiz_png/A0h5yD51CMaqqCEFpGbYmEIy7RIwXNrGxrEbqiczmnIz651Yemy0rSHqr9aG0nbmuknqiaJr4hm0794IoshHDd6Q/640?wx_fmt=png)

 Virt SIG 在即将发布的 openEuler 21.03 版本中，也给大家带来了很多新特性。在 QEMU+KVM 上，新增了热迁移 PRO，让虚拟机的迁移速度翻倍。同时增强 QEMU+KVM 的可靠性和可维护性。QEMU 将适配 Linux 内核 5.10 版本。

 

在 openEuler21.03 版本中，将发布增强版的 StratoVirt。增强版 StratoVirt 将在面对云原生轻量级虚拟化的场景中支持内存弹性，系统根据虚拟机的运行状态自动回收处于空闲状态的内存，进而提升虚拟机的资源利用效率与运行效率。在芯片架构上，StratoVirt 将支持 Arm、x86 芯片架构。支持 filebackend 。StratoVirt 也将在支持一些标准虚拟化的功能，例如支持大页、IOthread、IO-QOS 增强等等。
