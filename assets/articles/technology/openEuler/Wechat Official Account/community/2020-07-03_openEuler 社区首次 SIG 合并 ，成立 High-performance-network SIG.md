# [openEuler 社区首次 SIG 合并 ，成立 High-performance-network SIG](https://mp.weixin.qq.com/s/NKMdiXQsUUWpKSQ591-Hvw)

*openEuler 团队*[OpenAtom openEuler](javascript:void%280%29;)*2020-07-03 22:00:00*

经 openEuler 技术委员会批准，openEuler 社区正式成立High-performance-network SIG，完善 openEuler 在高性能网络场景的支持。openEuler 社区参与者和用户可获得由专业网络团队维护的稳定的高性能网络服务，并通过 openEuler 社区获得技术支持。

网络加速技术普遍采用 bypass 内核，offload 硬件等手段实现高性能网络服务，为上层应用提供多样化的高性能网络解决方案，目前网络加速技术已经被广泛应用在各种场景。为了进一步完善高性能网络解决方案，经过 openEuler 技术委员会批准，成立 High-performance-network SIG，该 SIG 组将会持续集成行业内广泛使用的高性能用户态协议栈，包含华为公司预计在 Q3 发布的一款posix语义兼容的轻量化高性能用户态协议栈。为广大用户提供简单易用，性能卓越的高性能网络服务。

同时，该 SIG 会提供 XDP、DPDK、RDMA 等网络加速技术及其相关的基础软件、驱动、语言库、性能工具、应用等软件包，其中包括一些知名网络类应用的维护，升级等工作，比如 cilium 等。

该 SIG 会持续追踪高性能网络解决方案的前沿和热点技术，包括但不限于：内核协议栈改造、智能网卡可编程、DPDK/RDMA/XDP 性能优化等。及时将行业网络的最新研究动态引入到 openEuler 社区，将 High-performance-network SIG 打造成网络领域的技术高地。

为了能够给 openEuler 社区参与者和用户带来更好的体验，以及聚焦业务方向，经 openEuler 技术委员会批准：

**同意 sig-dpdk与 sig-High-Performance-Network 合并，相关软件仓合并，相关软件仓的维护人等保持不变。**

这是 openEuler 社区成立以来首次 SIG 组的合并变动，两个 SIG 组的团队经过协商，一致认为合并更有利于该领域的发展，这充分展现了社区的自我演化，自我发展的能力。也体现了两个 SIG 团队高度的专业化能力。openEuler 技术委员会对此高度赞赏，并将持续支持该 SIG 组打造高性能网络的愿景。
