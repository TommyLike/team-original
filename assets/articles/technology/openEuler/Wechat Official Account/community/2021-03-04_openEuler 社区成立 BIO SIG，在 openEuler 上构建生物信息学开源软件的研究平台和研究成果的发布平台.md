# [openEuler 社区成立 BIO SIG，在 openEuler 上构建生物信息学开源软件的研究平台和研究成果的发布平台](https://mp.weixin.qq.com/s/q8jK9ElT5wluK_gIiP-Bew)

*周荔人*[OpenAtom openEuler](javascript:void%280%29;)*2021-03-04 18:05:23*

经 openEuler 社区技术委员会讨论决定，openEuler 社区正式成立 BIO SIG。BIO SIG 致力于发展 openEuler 上的生物信息领域的开源软件生态，在 openEuler 上构建一个生物信息学的研究平台和研究成果的发布平台。openEuler 社区通过建立 BIO SIG，希望让从事生物信息学的研究机构和人员，能够使用上 openEuler 提供的便捷、稳定的生物信息学相关的开源软件包。

BIO SIG 是 openEuler 社区中第一个跨学科的 SIG 组。体现了社区的多样性发展。

 

生物信息学是生命科学与计算机科学的交叉学科，利用生物学、计算机科学和信息技术处理大量复杂的生物数据以获取生物学信息。研究方向包含基因测序与功能发现、疾病基因诊断、蛋白质结构预测、药物设计与合成等等。比如说用于肿瘤治疗的靶向药，就是生物信息学的产物。随着生物信息量的爆炸增长和信息技术的快速发展，生物信息学是当今最具发展前景的学科之一。

 

生物信息领域将成为未来高性能计算和云计算的主要应用领域之一。

首先，生物信息领域前景广阔。2019年生物信息领域的市场达到86.1亿美元，根据 PharmiWeb 的预测，到2027年将会增长到247.3亿美元，复合年均增长率达 13.4%；

其次，生信研究需要强大的算力。以人类举例，46条染色体大概可以挖掘出数百 GB 的基因数据，而分析如此庞大的生物学数据，显然需要超高的计算能力。而云计算则为生信研究提供了开箱即用、随取随用的服务，并且方便了生信研究场景的复现和数据的共享。

 

生物信息学的发展离不开开源软件。目前，生物信息领域活跃着大量的开源软件，这些开源软件为生物信息学的各类问题提供了免费的解决方法，比如针对基因序列的比对问题，就有 BWA\\Bowtie2 等多款开源软件提供了不同算法的比对功能。大部分的生信研究都会用到开源软件。开源软件为生物信息学的软件基础设施做出了巨大贡献，加速了生物信息学的发展。

![](https://mmbiz.qpic.cn/mmbiz_jpg/A0h5yD51CMZePEAJCRd1leuiarAIBTWia5an4MeoMBZ7RLY1tOB5HDuKicoFeU0icqtHibf4pgvYZcqjDqbGYHmvUxA/640?wx_fmt=jpeg)

生信软件的安装适配是生信开发者面临的第一道难题。我们在openEuler 上安装生信软件时，出现了缺乏可靠的镜像源、缺少依赖库、依赖冲突、版本不一致等安装适配问题。

 

BIO SIG 建立的目的是希望在 openEuler 上构建一个生物信息学的研究平台和研究成果的发布平台。为了实现这些目标 BIO SIG 将从以下方面进行努力。

 

第一，  解决生物信息学开源软件在 openEuler 社区上的适配技术问题。提供安装简便，系统一致性好的生物信息学开源软件安装包。

第二，  构建基于 openEuler 的官方生物信息学开源软件仓库，提供丰富的生物信息学开源软件包，覆盖广大的生物信息学应用场景。

第三，  连接操作系统用户和生物信息学软件社区开发者，获取用户诉求，推动上游软件在 openEuler上的适配和演进。

BIO SIG 计划在2021年提供超过60款高频的开源生物信息学的开源软件包，使这些软件包具备服务高频次生物信息研究场景的能力。并在 openEuler 21.09 版本合入到主线当中。
