# [openEuler 社区成立 UKUI SIG](https://mp.weixin.qq.com/s/NoQpPKZvUtd2SEXV04RjAQ)

*openEuler 团队*[OpenAtom openEuler](javascript:void%280%29;)*2020-07-13 21:15:59*

经 openEuler 社区技术委员会讨论决定，openEuler 社区正式成立 UKUI SIG。UKUI SIG 的成立将进一步完善和丰富 openEuler 的桌面环境，openEuler 社区参与者和用户可获得由 UKUI SIG 维护的 UKUI 桌面环境，并且通过 openEuler 社区获得技术支持。

![](https://mmbiz.qpic.cn/mmbiz_png/A0h5yD51CMY4G079oT3v8akppxWwhXLnLqhPphsomicWKl2Z9ibecHD9QPiaYiaK1W0CVjp1wCU8UoLpooZ7Wib6eCg/640?wx_fmt=png)

UKUI SIG 首位的 Maintainer 为麒麟软件，是 openEuler 社区首批创始  SIG 之一。

***作为一个面向全球，支持多处理器架构，开放的操作系统开源社区，openEuler 一直以开放、热情的态度欢迎全球的开发者。我们很高兴能够加入 openEuler 社区，为社区的发展做出自己的贡献。同时，我们也希望通过参与 openEuler 社区，和社区一起发展，更好的为用户服务，为社区贡献更好的 UI 体验。在接下来的工作中，我们将继续进行 UKUI 的调优工作，增强 UKUI 的稳定性，认真对待社区的讨论和建议，在 openEuler 上提供一个稳定、易用的桌面环境***。UKUI SIG Maintainer 同样也是麒麟软件的高级工程师窦龑说到。

UKUI 是一款由麒麟软件主导开发的轻量级 Linux 桌面环境，其 3.0 版本按照简洁、易用、高效的设计理念，基于 QT 全新开发而成。

***omgubuntu.co*** 社区的知名软件评论员 Joey Sneddon 对 UKUI 3.0 给予了很高的评价，称其为 “Phenomenal”（非凡的）。

目前 UKUI 已经支持 Ubuntu、Debian、Arch 等主流的 Linux 发行版，即将支持 Fedora 和 openEuler。根据 SIG 的发布计划，支持 openEuler 的 UKUI 将于 7月底与大家见面。

麒麟软件作为中国开源操作系统的代表，一直都在积极的投身于各种开源社区中的工作，累计向开源社区贡献代码数百万行。

其中被 Linux、GNOME、Unity、OpenStack、Ceph 接收的 patch 超过 7000 个。麒麟软件加入 openEuler 社区是麒麟软件积极拥抱开源社区的又一次印证。

openEuler 社区坚信，通过和麒麟软件的合作，将会极大的繁荣 openEuler 的软件生态。为用户提供更加友好易用，简单轻松（Convenient and Intuitive）的用户界面。

目前 UKUI 对openEuler 的适配进度已完成80%，主要组件在 openEuler 上基本可以运行，代码已上传至 openEuler 社区，并且为社区提供了尝鲜版本。

**尝鲜版本下载链接**

***https://gitee.com/openkylin/ukui-issues/blob/master/Install.md***

目前主要的进行的工作是增强 UKUI 在 openEuler 上的运行稳定性，并将代码移植到 OBS 上进行编译和构造，以方便社区更好的集成和使用。

 

***「UKUI 是由老牌的开源企业麒麟软件发起，国内少有的正向开发的大型开源软件系统，也是在 openEuler 成立初期就设立的创始 SIG 组之一。这一切都表明麒麟软件和 UKUI 的开发团队抱着最大的诚意和 openEuler 共同构建开放，开源的社区生态，对此 openEuler 技术委员会高度赞赏，我们将一如既往的支持麒麟软件和相关SIG组在openEuler社区的健康发展。openEuler技术委员会对于即将发布的UKUI 3.0充满期待，并预祝UKUI在未来的版本中取得更大的成绩。」***

***—— 熊伟 openEuler 技术委员会委员***

 

 

**关于 openEuler**

openEuler 是一个开源、免费的 Linux 发行版平台，将通过开放的社区形式与全球的开发者共同构建一个开放、多元和架构包容的软件生态体系。同时，openEuler 也是一个创新的平台，鼓励任何人在该平台上提出新想法、开拓新思路、实践新方案。

openEuler 通过社区合作，打造创新平台，构建支持多处理器架构、统一和开放的操作系统，推动软硬件应用生态繁荣发展。

 

**关于麒麟软件**

麒麟软件拥有银河麒麟和中标麒麟两大操作系统品牌。包括服务器操作系统、桌面操作系统、嵌入式操作系统、麒麟云等产品，能够同时支持鲲鹏、飞腾、龙芯、申威、兆芯、海光等国产 CPU。麒麟软件注重核心技术创新，先后申请专利 329 项，其中授权专利 147 项，登记软件著作权 340 余项，主持和参与起草国家、行业、联盟技术标准 30 余项，荣获各类国家级、省部级和行业奖项 400 余个。产品通过了 CMMI5 级评估具有雄厚的科研能力和严格的管理规范。2018年，麒麟操作系统荣获国家科技进步一等奖。

目前，麒麟操作系统的软硬件生态经过20年积累，认证测试产品已经近万款。涵盖所有国内外主流软硬件和外设产品。同时，麒麟操作系统在党政、金融、能源、交通、医疗、教育、科研、制造领域广受用户信赖，目前服务的中国客户应超过10000家。根据赛迪顾问统计，麒麟操作系统产品，连续8年位列中国 Linux 市场占有率第一名。
