# [openEuler 社区成立 OSCourse SIG，在 openEuler 构建开源操作系统课程推广平台](https://mp.weixin.qq.com/s/6Nb5DbB3YSMPO7Hi7PYMQQ)

[OpenAtom openEuler](javascript:void%280%29;)*2021-07-12 19:11:24*

经 openEuler 社区技术委员会讨论批准，openEuler 社区正式成立 OSCourse SIG。OSCourse SIG 致力于发展和推动开源操作系统 Linux 的系统课程开发、维护、推广等工作。

基于当前的新兴硬件、新兴开源系统版本，以及最新操作系统内核技术发展路径，不断提供和完善开源操作系统课程内容。

openEuler 社区通过建立 OSCourse SIG，将包括 openEuler 在内的一系列开源操作系统教材、课件、课程、实训内容等操作系统教学资源，供全行业从事操作系统的研究机构和人员，以及有志于操作系统方向研究的学生、团体进行免费试用。

同时，通过 SIG 的成立，鼓励开源贡献者和其他院所、高校、公司等参与进这类教程的维护和拓展，为国内基础教育事业添砖加瓦。

OSCourse SIG 是 openEuler 社区中的第一个关于操作系统课程的 SIG 组。体现了社区的多样性发展。

操作系统位于底层硬件和应用软件之间，是每一个计算系统所必备的核心组件，是 IT 生态的核心环节，是 IT 巨头竞争的焦点，是软件领域的技术制高点，具有巨大的战略价值和商业价值。

随着人工智能、新型硬件的快速发展，如何适配当前从边缘计算的智能终端，到数据中心的云服务器，操作系统仍然是当今最重要也最核心的研究领域之一。

操作系统的发展离不开开源社区，事实上，Linux 操作系统作为主流的开源操作系统已经在数据中心计算领域成为主导地位。基于 Unix 变种的操作系统也在终端、智能设备、航空航天等科研、工业领域广泛使用。基于 Linux 的开源社区提供了丰富的开源软件可供使用。一款新的硬件或设备的出现，第一件事总是通过移植操作系统来为硬件提供丰富的应用场景。

因此，持续不断的为操作系统领域提供开源贡献者，引领国内高校学生、从业者不断的关注最新的操作系统技术发展，成为了维护国内开源操作系统社区的一个关键命题。

OSCourse SIG 建立的目的是希望在 openEuler 社区中，构建一个开源操作系统教程的维护平台。为了实现这些目标 OSCourse SIG 将从以下方面进行努力。

第一， 提供丰富的、模块化的教学体系设计，提供一套、乃至多套操作系统教程，通过灵活搭配组合，适应不同人群的教学需求。 

第二、SIG 组将提供开源的课程教材、课件、实训内容、参考代码等，致力于保证学习者能通过课程实践，掌握当前开源操作系统的开发能力，并培养操作系统开源贡献者。

第三、SIG 将先以 openEuler 操作系统发行版为最先的操作系统教程项目进行维护。不断开发并维护 openEuler 和鲲鹏硬件特性相关的操作系统课程内容。并进一步收录和维护 RISC-V、ARM、X86 等不同体系架构和新型硬件特性相关的操作系统教程。 

第四、SIG 将连接操作系统课程学习者、开源社区贡献者、国内外高校教学人员，获取用户诉求，推动操作系统教程的是配合和演进。

OSCourse SIG 计划在 2021 年提供超过 140 课时的操作系统教程，包括课件、实训内容等。
