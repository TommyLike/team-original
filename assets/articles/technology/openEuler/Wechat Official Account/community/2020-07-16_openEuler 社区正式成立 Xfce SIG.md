# [openEuler 社区正式成立 Xfce SIG](https://mp.weixin.qq.com/s/sLypFcevjeByQe5Q5kLWdQ)

*openEuler 团队*[OpenAtom openEuler](javascript:void%280%29;)*2020-07-16 19:30:00*

日前，openEuler 社区正式成立 Xfce SIG，旨在完善和丰富 openEuler 操作系统的桌面环境，openEuler 社区参与者和用户可获得轻量级桌面环境  Xfce 最新版本，并通过 openEuler 社区获得技术支持。

Xfce 是类 UNIX 操作系统上的轻量级桌面环境，它致力提供一个快速与低资源消耗，并且具有视觉吸引力、易于使用的桌面环境。Xfce 桌面环境包含计算机桌面环境的全部功能，但底层由独立的软件组件构成，可根据需要单独使用。

为了进一步完善 openEuler 对 Xfce 的适配，丰富 openEuler 的桌面功能，经过 openEuler 技术委员会讨论，正式成立Xfce SIG，作为首位 Maintainer 的操作系统厂商拓林思软件（TurboLinux），将承担主要适配工作。

拓林思计划在2020年10月以前完成 Xfce 桌面环境的适配工作。

 

***一般国内的 Linux 发行版本都是基于 Debian/Ubuntu 或 Fedora/CentOS 的衍生版本，衍生版本的所谓建立生态本质上是在帮助原生版本扩展生态。而 openEuler 从源头做起，可能是网聚最多国内上游社区（upstream）开发者的社区，并且我认为 openEuler 有望像 Debian 和 Fedora 那样成为世界最主流的版本之一。因此我们很高兴加入 openEuler 社区，为 openEuler 社区贡献一份力量。除了目前参与的 Xfce 桌面环境之外，还将为完善和丰富 openEuler 的数据库生态而努力。”***

***——陈棋德 北京拓林思软件有限公司副总经理***

***首先欢迎拓林思加入 openEuler 的大家庭，拓林思是老牌的 Linux 发行厂商，为国内的 OS 领域培养了大量人才。这次拓林思带领 Xfce SIG 组，将这个久负盛名的轻量化桌面带到了 openEuler。我们期望和拓林思一起，再次在企业市场展示拓林思深厚的积累和品牌效应。同时，拓林思作为国内 Linux 行业的先驱，我们也希望它将行业经验、开源文化融入到 openEuler 社区，帮助 openEuler 成为行业领先的开源社区和 Linux 平台。***

***——熊伟 openEuler 社区技术委员会委员***

      

        

**关于openEuler**

 

openEuler是一个开源、免费的 Linux 发行版平台，将通过开放的社区形式与全球的开发者共同构建一个开放、多元和架构包容的软件生态体系。同时，openEuler 也是一个创新的平台，鼓励任何人在该平台上提出新想法、开拓新思路、实践新方案。

 openEuler 通过社区合作，打造创新平台，构建支持多处理器架构、统一和开放的操作系统，推动软硬件应用生态繁荣发展。

      

**关于拓林思**

 

拓林思于 1999 年 3 月发布了 TurboLinux 3.0.2，这是国内第一个 Linux 中文化版本，开创了 Linux 中文化先河。TurboLinux 是世界公认的 Linux 中文化方面的领导者和奠基者。

拓林思完成了众多企业级项目实施和服务，拥有丰富的成功范例和经验，客户遍布金融、电信、能源等各个行业。
