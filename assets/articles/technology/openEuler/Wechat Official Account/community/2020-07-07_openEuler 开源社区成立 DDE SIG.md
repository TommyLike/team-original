# [openEuler 开源社区成立 DDE SIG](https://mp.weixin.qq.com/s/kOHrauS-gK_YxYWrrR-hJg)

*openEuler 团队*[OpenAtom openEuler](javascript:void%280%29;)*2020-07-07 21:16:08*

日前，经过 openEuler 技术委员会讨论决定，openEuler 开源社区正式成立 DDE SIG。DDE SIG 的成立将进一步完善和丰富 openEuler 的桌面环境，openEuler 社区参与者和用户可获得由专业团队维护的最新桌面环境 DDE 版本，并通过 openEuler 社区获得技术支持。DDE SIG 的首位 Maintainer 为操作系统厂商统信软件。

 

DDE(Deepin Desktop Environment) 中文名称为深度桌面环境。是统信软件自研的一款全功能的桌面环境。为用户提供美观易用、智能协同的界面交互。目前，DDE 已经被移植到包括 Archlinux、OpenSUSE、Ubuntu、LinuxMint、Fedora、Manjaro 和 Anthon 等主流 Linux 发行版中。

目前，统信软件已经将最新版 DDE 集成至统信 UOS Server Euler 中，统信 UOS Server Euler 以 openEuler 20.03 LTS（Linux kernel 4.19）社区发行版为基础，通过整合自研 DDE 桌面环境和图形界面的应用程序，第三方软件以及相关开源软件等内容，形成针对常规、云、虚拟化、容器、大数据等应用场景的通用操作系统方案。

作为操作系统领军企业，统信软件拥抱开源，贡献开源。未来，统信软件将和众多开发者一起全力支持构建 openEuler 开源社区。

*openEuler技术委员会非常高兴统信软件将行业内知名的 DDE引入到 openEuler 社区，这充分展现了国内开源企业高度的开放性和合作性。我们相信，秉持开放合作的社区精神，openEuler 社区和统信软件一定能共同成长，共同推动国内开源行业的产业化进程。*

***——熊伟 openEuler 社区技术委员会委员***

 

*openEuler 集合全国乃至全球开源贡献者，共同构建高效、稳定、安全的开源操作系统，帮助用户快速构建数据库、大数据、云计算、人工智能等高价值应用场景。我们很高兴加入 openEuler 社区，统信软件致力于打造符合国人习惯的国产桌面系统，我们希望通过 DDE 桌面系统与 openEuler 社区的友好链接，进一步丰富社区软件生态系统，在未来能够更快更好的满足国内外用户多种应用场景诉求。*

***——叶青龙 统信软件技术有限公司项目经理*** 

**关于 openEuler**

openEuler 是一个开源、免费的 Linux 发行版平台，将通过开放的社区形式与全球的开发者共同构建一个开放、多元和架构包容的软件生态体系。同时，openEuler 也是一个创新的平台，鼓励任何人在该平台上提出新想法、开拓新思路、实践新方案。  
openEuler 通过社区合作，打造创新平台，构建支持多处理器架构、统一和开放的操作系统，推动软硬件应用生态繁荣发展。

**关于统信软件**

统信软件技术有限公司由国内多家长期从事操作系统厂商整合而成，以“打造中国操作系统创新生态”为使命，致力于为不同行业的用户提供安全稳定、智能易用的操作系统产品与解决方案。

统信软件的代表产品统信 UOS 基于 Linux 内核采用同源异构技术打造，同时支持四种 CPU 架构（X86-64、ARM64、MIPS64、SW64）和鲲鹏、龙芯、申威、海光、兆芯、飞腾、Intel/AMD 等 CPU 平台，能够提供高效简洁的人机交互、美观易用的桌面应用、安全稳定的系统服务。
