# [IDE-SIG在社区成立：构建全球开发者工具链生态，赋能嵌入式行业升级](https://mp.weixin.qq.com/s/-09iGPSbPVLwNxctdgTinQ)

原创*openEuler*[OpenAtom openEuler](javascript:void%280%29;)*2024-07-25 17:52:08北京*

经OpenAtom openEuler（简称"openEuler"）社区技术委员会讨论决定，IDE-SIG在社区成立。该组由**华为技术有限公司、粤港澳大湾区国家技术创新中心**和**蚂蚁科技集团股份有限公司**发起，联合江苏润和软件股份有限公司、深圳开鸿数字产业发展有限公司、深圳行云创新科技有限公司等国内多家领先科技企业共同设立。

随着智能制造的快速发展和 openEuler 的影响力不断扩大，嵌入式系统开发需要一个易用、好用的IDE（集成开发环境，Integrated Development Environment）来支撑快速便捷的开发工作。**IDE-SIG为openEuler 生态系统量身打造，旨在为开发者提供一套更为高效、优质的工具链解决方案，加速业务开发迭代进程，推动嵌入式行业更广泛的技术应用与创新，进而增强嵌入式软件在全球市场的竞争力。**我们诚邀全球开发者加入IDE-SIG，携手推进 IDE 技术的前沿探索，共同塑造高效开发的未来蓝图。

**专业级、多平台兼容、全功能IDE**

我们旨在构建一款专业、直观且高效的嵌入式IDE，该环境支持多种硬件平台和编程语言，配备全面的开发工具和调试功能，为嵌入式系统各类开发者提供一站式、低门槛、多模态的开发平台，确保高效流畅的研发体验。实现资源分配统一化、计算能力远程化、开发服务智能化、工作空间安全私有化、开发环境准备自动化、用户界面轻量化；基于安全可信、自主可控、共建共享、端云一体的研发底座，建立嵌入式系统工具链及关键标准件（含操作系统）的开发环境集成标准。

![](https://mmbiz.qpic.cn/mmbiz_jpg/jqTxnpIZ2mCL2mPnkzmict5mgj9Y9Ou37Q4QNjQiat0ZOf8SZibzRLWw62jdQetv2aqoyAHFqicoNuWkwDHMbM7Hgg/640?wx_fmt=jpeg&from=appmsg)

▲嵌入式IDE框架

嵌入式IDE框架基于 TypeScript 和 React 进行编码，实现了包含资源管理器、编辑器、调试、仿真、Git 面板、搜索面板等核心功能模块，开发者只需要进行简单的配置，就可以快速搭建属于自己的本地或云端 IDE 产品。可支持Web、Electron、纯前端三种模式灵活选择。另有嵌入式软实时、混合部署、硬实时插件供社区开发者免费使用。IDE以OpenSumi开源框架为基座，实现功能模块化、易扩展和定制性的需求，同时依托OpenSumi研发团队的支撑，推进社区生态建设。

**专业高效的开发工具，打造共赢生态**

**?        提供专业工具**

IDE开发工具面向企业和个人开发者提供工程创建、开发、编译、调试、仿真、烧录等嵌入式开发全流程全家桶，帮助开发者迅速构建自定义开发环境，提供定制化、易扩展的开发工具链以满足特定开发场景。创建一个IDE基础框架源码仓，向社区所有用户开源，联众创新，持续优化并拓展框架功能。

**?        共筑嵌入式人才高地**

组织专业培训、学术论坛及技术交流会，深化与高校及科研机构的协作，吸引更多开发者加入openEuler社区，促进社区的多样性和活跃度。同时，补充国内嵌入式领域的人才培养方式，促进知识与技术的接力传递，为行业输送源源不断的高素质专业人才。

**?        产业链协同合作**

IDE-SIG组织将在促进嵌入式产业链上下游企业间紧密协作方面，发挥重要的作用。它不仅是产业上下游企业合作的桥梁，更是推动整个产业蓬勃发展的强大引擎。借助这一平台，各合作伙伴能够开启更广阔的合作空间，挖掘嵌入式技术领域的深层商业价值与市场潜力，促进市场生态的多元化与可持续发展。
