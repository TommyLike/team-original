主题：**Empower BigData on Arm64**

![](https://mmbiz.qpic.cn/mmbiz_png/jqTxnpIZ2mCBE9VIDicbvLXwlzbwxo7vD4wbj1PtWTZUE3iaqcBTgZz8NxzST8z2BFIBAZVmgq11BGZE9wsTNj3w/640?wx_fmt=png&from=appmsg)

**顾煜祺**

Staff software Engineer, Arm China  
Apache Bigtop PMC member, committer  
Apache Ambari PMC member, committer

**?????** 议题简介：

本议题将概述大数据项目生态在Arm平台上的应用和性能情况。将简要介绍在Apache项目：Apache Bigtop和Apache Arrow上的应用和优化工作。我们也将探讨Spark SQL加速方案：Gluten和Velox，并分享Velox在Arm平台上的一个优化案例，以及一些针对大数据工作负载的Arm性能分析工具。

![](https://mmbiz.qpic.cn/mmbiz_png/A0h5yD51CMbzheJx0QH7NdQial0iaee2icqhhicCSU9UctP0XLXqj5sbXibhTWZSVFf1NlExMvs95zzjgSgxnJcBVww/640?wx_fmt=other&wxfrom=5&wx_lazy=1&wx_co=1)

主题：**UADK加速大数据：鲲鹏服务器的关键应用场景**

![](https://mmbiz.qpic.cn/mmbiz_png/jqTxnpIZ2mCBE9VIDicbvLXwlzbwxo7vDChWuADicMc3bWZPsnm9vcu2gJ1qIwdOocE6SKZMsiaq07fib2WRq0SOyg/640?wx_fmt=png&from=appmsg)

**徐国栋**

Senior Tech Lead

Linaro Ltd.

**?????** 议题简介：

本议题将为您介绍“UADK加速大数据”项目，这是华为鲲鹏服务器的关键应用之一。我们将探讨在大数据领域使用UADK时的集成挑战，并分享Linaro着陆团队在大数据全栈软件模块集成方面的成果。此外，我们还将详细讨论UADK在加速HDFS Transparent Encryption在鲲鹏920上的性能优势，以及实测的TeraSort/HDFS透明加密（SM4-CTR）的处理速度提升。让我们一起深入了解这个令人兴奋的话题！

![](https://mmbiz.qpic.cn/mmbiz_png/A0h5yD51CMbzheJx0QH7NdQial0iaee2icqDrrABmfA2nNpZWRyQbrgwPCibibq8gLH2xW8sHrFQwhzaQsoma789fRA/640?wx_fmt=other&wxfrom=5&wx_lazy=1&wx_co=1)

主题：**Bigtop Manager：面向云原生的大数据集群管理平台**

![](https://mmbiz.qpic.cn/mmbiz_png/jqTxnpIZ2mCBE9VIDicbvLXwlzbwxo7vDDV7XFL5HX0edp1gs29UMSIicyN8HYQmEVjRAKEmZb58zKFicQypHhUhw/640?wx_fmt=png&from=appmsg)

**吴治国**

Apache Ambari PMC

**?????** 议题简介：

Bigtop Manager是一款现代化的，面向云原生的大数据集群管理平台，提供了一套轻量级的方案高效地管理bigtop所支持的大数据组件。经过在openEuler社区BigData SIG孵化，6月份毕业进入Apache社区。今天主要分享Bigtop Manager的立项背景，当前进度，架构设计以及未来规划等。

![](https://mmbiz.qpic.cn/mmbiz_png/A0h5yD51CMbzheJx0QH7NdQial0iaee2icqRMl9BcxPSEWicy2YiaCXhCwk5ddhZgNDqnVQUxlCG6HFgcQHUEylra8w/640?wx_fmt=other&wxfrom=5&wx_lazy=1&wx_co=1)

主题：**一种高效的Spark SQL Native引擎实现方案**

![](https://mmbiz.qpic.cn/mmbiz_png/jqTxnpIZ2mCBE9VIDicbvLXwlzbwxo7vDdlcWUOwYSwMhPzytYE08wgib6DNIMrKy3Cx4Zu6wB9JVvFxTzuUzV2A/640?wx_fmt=png&from=appmsg)

**陈强**

Apache Bigtop Commiter  
openEuler Bigdata SIG maintainer

**?????** 议题简介：

本议题将介绍一种C++实现的Spark SQL Native引擎解决方案，基于Spark提供的Extension功能插件式实现无需侵入修改代码。动态代码生成和静态编译结合并使用LLVM动态优化生成代码，算子自动向量化，提升Spark SQL的性能。
