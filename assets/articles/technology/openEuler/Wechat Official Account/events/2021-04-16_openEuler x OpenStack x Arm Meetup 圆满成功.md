# [openEuler x OpenStack x Arm Meetup 圆满成功](https://mp.weixin.qq.com/s/sgCWEQyGnPtWrUiq_Ao0Zw)

*郑振宇*[OpenAtom openEuler](javascript:void%280%29;)*2021-04-16 17:54:13*

4 月 10 日，由 openEuler 社区主办、联通数科与 EasyStack 协办的 openEuler OpenStack SIG PTG 暨 openEuler x OpenStack x Arm Meetup 在北京中关村创业大街成功举办。活动聚集了来自 17+公司及机构的 60+资深专家及开发者，围绕 openEuler 与 OpenStack 的融合、Arm 软件生态发展及落地挑战等话题展开了深入的探讨。

![](https://mmbiz.qpic.cn/mmbiz_jpg/A0h5yD51CMYsiaoZvjastLfAJMB87icVKTYDya1lJofvxyJicibvmNcxWepa2kcybOXY2CKeXr9ic6ykLrPoFfy2nibA/640?wx_fmt=jpeg)

本次活动分为上午、下午两个部分，10 日上午举办的是 openEuler 中 OpenStack SIG 的首次 PTG(Project Team Gathering)活动，10 日下午举办的是 openEuler x OpenStack x Arm Meetup。

## OpenStack SIG PTG

在 2021 年 3 月 31 日发布的 openEuler 21.03 创新版本中，来自联通数科、麒麟软件、华为、Linaro 等公司的开发者所组成的 openEuler OpenStack SIG 成功将 OpenStack 的最新版本 Victoria 引进到 openEuler 软件仓库，并进行了完整的 Tempest 测试（核心组件全量部署端到端测试），为 openEuler 在云计算领域的生态拓展提供了技术基础，也为构建开放可靠的云计算技术栈提供了最佳实践。随着 21.03 版本的正式发布，OpenStack SIG 在下一个发布周期的工作规划就被提上了议程，这也是本次活动举办的初衷。

SIG(Special Interest Group)是社区根据领域划分的各个领域的项目组/兴趣小组，openEuler 社区按照不同的 SIG 来组织，以便于更好的管理和改善工作流程。openEuler 中 OpenStack SIG 的愿景是为用户提供开放可靠的云计算技术栈，OpenStack SIG 目前已成为 openEuler 社区中最活跃的 SIG 组之一。

PTG 活动是从 OpenStack 社区借鉴而来，PTG 活动将分散在各地的 SIG 开发者、用户聚集在一起，秉承 Open Source、Open Design、Open Development、Open Community 的四个 Open 理念，为大家提供面对面深度交流的机会。

在本次 PTG 活动中，来自联通数科、Open Infrastructure Foundation、Linaro、Arm、华胜天成、天津卓朗科技、中国电信、浪潮、烽火、建设银行建信金融科技有限责任公司、华为等公司的开发者对 OpenStack SIG 在 openEuler 下一个发布周期内(20.03 LTS SP2 和 21.09)的重点工作进行了深度的讨论。

![](https://mmbiz.qpic.cn/mmbiz_jpg/A0h5yD51CMYsiaoZvjastLfAJMB87icVKTIjvgoZJIYXAF2nMO5fsIHU87Mvuo2VI8pFBb9P2nfSUzbjWUH7YRxg/640?wx_fmt=jpeg)

PTG 会议讨论采用 OpenStack SIG Maintainer 引导、开放议题收集、开放讨论的形式开展，通过深入的交流，OpenStack SIG 最终在以下几个方面达成了一致：

1. SIG 运作规范、软件包 Spec 规范、问题及需求反馈流程规范
2. openEuler 在 OpenStack 社区上游的 CI 适配推动策略
3. openEuler 20.03 LTS SP2 中 OpenStack 的版本选择，及相关依赖的版本匹配
4. OpenStack openEuler LTS 版本升级路径

同时，OpenStack SIG 对在场的用户和开发者进行了需求收集，与会者对 OpenStack 容器化部署、集群部署、加速设备支持、高级网络组件测试覆盖、混合架构部署等各方面提出了诉求，OpenStack SIG 将会在后续的双周会议中这些需求进行更为细致的讨论。

## openEuler x OpenStack x Arm Meetup

近几年来，随着基于 Arm 架构的处理器在数据中心领域的快速发展，以及软件生态的快速丰富，越来越多的用户开始考虑将基于 Arm 架构的硬件产品引入到自己的数据中心中。在引入过程中，往往涉及从底层硬件到操作系统，再到云基础设施层，以及最终用户直接接触到的业务软件等整个软件栈的适配及调优的过程。在 10 日下午的 Meetup 中，来自联通数科、EasyStack、Linaro、华为等公司的专家就围绕 openEuler、OpenStack、Arm 软件生态等话题为大家带来了精彩的主题分享。

现场首先由 Open Infrastructure 基金会（原 OpenStack 基金会）中国社区经理李昊阳做开场致词，李昊阳对 openEuler OpenStack SIG 推动 OpenStack 在国产操作系统以及 Arm 架构上的适配、调优工作表示了认可，并且很高兴看到 OpenStack 社区中 PTG 的讨论方式能够为 openEuler 社区的发展带来实际价值。后续他也将会帮助搭建 openEuler 社区与 Open Infrastructure 基金会沟通的桥梁，也希望 openEuler OpenStack SIG 能够与 OpenStack 社区 Multi-Arch SIG 多交流、多合作，共同打造开放可靠的云计算技术栈。
