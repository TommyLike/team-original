# [第11届开源黑客松成功举办：开源让世界更美好](https://mp.weixin.qq.com/s/yZZIyQAHl1leeOCEF7f4zg)

黑客松是烟火气浓重的代码集市。

黑客松作为开源开发者搭建交流沟通的互动平台，以实现 bug 修复和代码优化为目标，推动开源技术广泛使用。活动自开办以来，始终坚持“开放代码、开放设计、开放开发、开放社区”的开源理念，用实际编程行动鼓励开发者技术创新和创意实践。第11届开源黑客松覆盖 openEuler、Mindspore、openGauss、openLooKeng、KubeEdge、OpenStack、K8S、SODA、Kata、Ceph、ChubaoFS 、Celadon 等十几个项目，活动参与者超百人规模。

![](https://mmbiz.qpic.cn/mmbiz_jpg/A0h5yD51CMbeFqklfP1EScJSOian0ia7JQKkykODdicKOicLV2FgIGJl5JHwdqDj1cNLW1tPQ565ycNia5tmHiamUmqw/640?wx_fmt=jpeg)

黑客松是工程师们使用代码这种社交货币相聚在一起，以他们想要的方式，去做他们想做的事，解决他们想解决问题的一种形式。

![](https://mmbiz.qpic.cn/mmbiz_jpg/A0h5yD51CMbeFqklfP1EScJSOian0ia7JQG5E6LKFmckYmqmpP94yiasxgBcGibLJCDbQcluUBYlAPCzTBSqXqGtzg/640?wx_fmt=jpeg)

第 11 届开源黑客松成功闭幕。118 名工程师相聚马栏山，就是谢娜说的那个“我是马栏山、马栏坡、马栏镇、马栏村的马小姐”的那个马栏山，进行了为期3天的黑客松活动。

这是一场跨项目、跨社区、跨公司的很工程师 style 的活动。代码进行时是开源黑客松活动的主题。

![](https://mmbiz.qpic.cn/mmbiz_jpg/A0h5yD51CMbeFqklfP1EScJSOian0ia7JQNJaNiajUMRYdOhABZeE9sicSWiaeamMHGclcllu93CBKabWoovqhjhmpA/640?wx_fmt=jpeg)

用户在 openEuler 安装 openGauss 的时候，会觉得安装过程有那么亿点点问题。用户可以找到 openEuler 的工程师交流一下，看这个问题怎么解决。大家身处不同项目，单又分工明确又密切协作，这是黑客松的独有的魅力之一。

![](https://mmbiz.qpic.cn/mmbiz_jpg/A0h5yD51CMbeFqklfP1EScJSOian0ia7JQ4skb0EXBicfGgAibfPI3h3RORytv0bPicicicVy4rbyyic6IBOWMiagSFZlAA/640?wx_fmt=jpeg)

不同社区的项目，在技术实现上虽然略有不同，但是却面临着相同的技术问题。项目只是大家对同一问题不同解决方式的体现。

![](https://mmbiz.qpic.cn/mmbiz_jpg/A0h5yD51CMbeFqklfP1EScJSOian0ia7JQz5KcQiacvPb5KufDHqvT9NbibZryzAcia1Q590n9Xps7vaBll2h88V4qA/640?wx_fmt=jpeg)

沟通能够更好的解决问题。希望下一届黑客松能够有更多不同的社区的工程师，跨越社区之间的 Gap，加入到这场活动中。KEEP CALM AND CODE ON。

黑客松的另外一个魅力，就是用代码展示你是谁，至于你是哪个公司的，在会场中并不那么重要。大家都是用 26 个英文字母来解决问题，从“基础架构”层面做到了统一，所以整个交流会无比轻松愉快，甚至会感觉有种相见恨晚的感觉。
