# [欢迎兆芯加入 openEuler 社区](https://mp.weixin.qq.com/s/4dh1E-XG0YugESKjihecMg)

[OpenAtom openEuler](javascript:void%280%29;)*2021-03-01 22:25:24*

2021年3月1日，兆芯签署 CLA ，正式加入 openEuler 社区，成为继鲲鹏、飞腾之后，第三个加入 openEuler 社区的芯片厂商。

![](https://mmbiz.qpic.cn/mmbiz_jpg/A0h5yD51CMYZmQ8OhTYMTl5eAUmnnteaI1sBDicuoqpiaKThPQVWWGo5LfUd5y3sc5nwNQBIViczpzYUaHJOWSqicA/640?wx_fmt=jpeg)

CLA 是 contributor license agreement 的缩写，中文为“贡献者许可协议”。

今天 openEuler 社区迎来 x86 架构的芯片厂商代表——兆芯。在今后兆芯与 openEuler携手，进行更多的技术实践与探索。

实现多种计算架构系统运行，充分释放不同架构计算能力，操作系统起到承上启下的重要作用。

openEuler 作为一个支持多样性算力的操作系统，支持 x86、Arm、RISC-V 等多种处理器架构。

鲲鹏和飞腾作为 Arm 架构的厂商代表已经加入 openEuler。

在 RISC-V 架构上，中科院软件所提供了基于 RISC-V 架构的 openEuler 已经成功运行在果壳处理器上。

openEuler 社区欢迎更多芯片厂商加入，共同构建最具活力的开源社区。
