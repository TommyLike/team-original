# [生态满足度+1｜同方完成 openEuler 兼容性认证，共同推动行业生态发展](https://mp.weixin.qq.com/s/RJI0qTbywzxmEAcpbobPtQ)

[OpenAtom openEuler](javascript:void%280%29;)*2021-05-25 18:02:35*

![](https://mmbiz.qpic.cn/mmbiz_jpg/A0h5yD51CMZsDTyEea8puqefqaEFfPDwP2XP63BqHeIRLbYgasSjEjoTce2Tw7wXJLicholgVmYZ5sEo42njyEQ/640?wx_fmt=jpeg)

2021年初，鲲鹏整机伙伴同方计算机有限公司（简称“同方”）正式签署 CLA（Contributor License Agreement，贡献者许可协议），加入 openEuler 社区。

本月，同方超强服务器 K620、K640 完成与 openEuler 的兼容性认证。

![](https://mmbiz.qpic.cn/mmbiz_png/A0h5yD51CMZsDTyEea8puqefqaEFfPDwI9G7HEwIrticZNNINbCno0QtK9ibrO3cEqWTIDFJLdOfPZaicODaAMkEg/640?wx_fmt=png)

在此次适配测试中，基于鲲鹏硬件平台的同方超强服务器 K620、K640 服务器与 openEuler 20.03 LTS SP1 进行了兼容性测试、稳定性测试和性能测试，测试结果表明：同方超强 K620、K640 服务器与 openEuler 20.03 LTS SP1 完全兼容，整体运行良好，稳定可靠，性能优异。

同方超强服务器 K620、K640 系列被广泛适用于通用服务器使用场景和云计算、虚拟化、数据库、大数据、人工智能、冷热数据存储、视频监控、云存储等应用环境，将超强算力引入每一个数据中心，满足各行各业数据中心多样性计算、绿色计算的需求，还可支持卫星遥感、气象预测、地质勘探、基因测序等高精尖领域的高性能计算，能够胜任关键行业信息系统中高端芯片的应用需求。

未来，同方将积极携手 openEuler 社区与合作伙伴，共同促进产业链协同创新发展，与行业内的众生态伙伴共建创新平台，推动行业生态繁荣发展。同时进一步完善行业生态适配工作，为用户提供更丰富完善的产品与服务，从而帮助用户更好地进行数字化、网络化及智能化转型。
