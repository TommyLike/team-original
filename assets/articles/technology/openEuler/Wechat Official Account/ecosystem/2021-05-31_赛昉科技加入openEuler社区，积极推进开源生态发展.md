# [赛昉科技加入openEuler社区，积极推进开源生态发展](https://mp.weixin.qq.com/s/KkpsotexCz7th_SQ1vovdg)

[OpenAtom openEuler](javascript:void%280%29;)*2021-05-31 18:00:00*

近日，业界领先的 RISC-V 处理器、平台及解决方案提供商赛昉科技有限公司（StarFive，以下简称“赛昉科技”）签署 CLA（Contributor License Agreement 贡献者许可协议），正式加入 openEuler 社区。

![](https://mmbiz.qpic.cn/mmbiz_png/A0h5yD51CMZDo8Z1ciaiaibBIsdkl7dibjlT3pyNQ3UjkibBsKap2NBLtGtUQKTTjsmuX3rGAmZ7d51OaDmQExThuDA/640?wx_fmt=png)

RISC-V 作为一款通用处理器指令集架构（ISA），具有开源、精简、可扩展能力强、设计协同性能优越等技术优势。目前，openEuler 已实现了对于 RISC-V 架构的支持，并于 2019 年成立 RISC-V SIG（Special Interest Group）组，致力于为 openEuler 构建面向 RISC-V 架构操作系统、丰富 RISC-V 软件生态。

赛昉科技的技术源于 RISC-V 开源架构，具有雄厚的 RISC-V 软硬件及平台化的技术实力。作为 RISC-V 国际协会高级会员 、国内开源生态的积极参与者和推动者，公司始终秉承源自开源、忠于开源、贡献开源的理念。本次加入 openEuler 社区，**标志着赛昉科技推动国内 RISC-V 生态发展进入了快车道**，未来赛昉科技将积极参与社区合作，与广大开发者及生态伙伴共同推进软硬件协同及云边端协同的创新设计，并参与到 SIG 的联合研发项目中，共建覆盖全栈式全场景的开源创新平台，促进国内生态的繁荣发展，共同创造并分享开源生态所带来的创新价值。

#### 关于赛昉科技：

赛昉科技有限公司（StarFive）成立于 2018 年，是中国 RISC-V 技术和生态的领导者，拥有完整的、经过硅验证的 RISC-V CPU IP 产品线和平台化的软硬件全栈式芯片解决方案。赛昉科技分别在中国、广东顺德、北京、成都、台湾新竹和马拉西亚设立了研发和销售中心。**目前已相继发布了全球首款基于 RISC-V 星光系列 AI 单板计算机、全球首款基于 RISC-V 的惊鸿 7000 系列人工智能及图像处理平台、全球首款全指令高性能 RISC-V CPU IP 天枢系列处理器**，产品广泛应用于智能家电、智能监控、工业机器人、交通管理、智能物流、穿戴设备、固态存储、网络通信、边缘计算等领域。更多信息，敬请访问：www.starfivetech.com
