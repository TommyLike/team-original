# [「转」进迭时空加入 openEuler 社区，openEuler 内核主线已完成对 SpacemiT K1 的基本支持](https://mp.weixin.qq.com/s/Ojlj5S2faBYHCLh1m7-CyA)

*OERV 宣传小队*[OpenAtom openEuler](javascript:void%280%29;)*2025-04-03 18:46:36广东*

近日，进迭时空科技有限公司，签署 OpenAtom openEuler（简称“openEuler”） 社区 CCLA（Corporate Contributor License Agreement 企业贡献者许可协议），正式加入 openEuler 社区。

![](https://mmbiz.qpic.cn/mmbiz_jpg/jqTxnpIZ2mBwtobTfLmdicnZiaTs0YHChdrict3ah6FyquMhq4HbKDb7Wp1APev4KelqC3w7lWgVfWRV5cDae8kHw/640?wx_fmt=jpeg&from=appmsg)

进迭时空是一家基于新一代 RISC-V 架构的计算生态企业，布局高性能 RISC-V CPU 核、RISC-V AI 核、AI CPU 芯片、NOC 总线、软件系统等全栈计算技术，提供端到端的计算系统解决方案。秉持进取不息、迭代不止的创业心态，致力于以 RISC-V 新 CPU 构建面向大模型新 AI 时代的最佳原生计算平台，助力 AI PC、AI 机器人等新应用的发展。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/tkVlnC85sFEJAEdvichuGGefXuhoDeb9YiakuHlX8qiaCJMo25LFrZPAwLFRY2BAo7CzOdCCMZRL6ceY0vicPS1cMA/640?wx_fmt=jpeg&from=appmsg)

目前，进迭时空与软件所 OERV 团队合作，已经完成了 openEuler 内核 6.6 版本主线对 SpacemiT K1 的基本支持，主要包括对设备树（DTS）、时钟（CLK）、复位控制器（reset）、引脚控制（pinctrl）、GPIO 子系统以及 UART 串口的支持。相关工作通过 OERV 内核同源工程完成，并在 OERV 小组的指导下达到合入 OLK6.6 分支的质量要求。openEuler 对 SpacemiT K1 完整支持将在 2025 年 9 月完成，将随 openEuler 25.09 正式发布。

未来三年，进迭时空将与 OERV 携手共同建设 openEuler 社区，依托 openEuler 的开源操作系统生态，重点在以下技术方向持续贡献：

1. 深化技术合作：积极参与 openEuler RISC-V SIG 建设，为社区定期贡献 OS 以下的软硬件解决方案。
2. 共建生态标准：联合生态伙伴，推动建立 RISC-V 服务器底座及操作系统技术规范与行业标准。
3. 产业协同发展：联合产业上下游合作伙伴，完成 RISC-V 高性能计算解决方案落地，推进产业发展。
4. 共建繁荣生态：双方将持续探索操作系统及高性能计算硬件底座协同创新，共建 RISC-V 高性能计算生态圈的繁荣。
