# [神策数据加入 openEuler 社区，推动开源生态繁荣发展](https://mp.weixin.qq.com/s/n0M-rx2vpmR8y14BcRM87w)

[OpenAtom openEuler](javascript:void%280%29;)*2021-07-07 18:05:00*

![](https://mmbiz.qpic.cn/mmbiz_jpg/A0h5yD51CMZ3rNPzl2Wuz2lyFPlbibcUuticUsNY2Mc3gKqkkpSPknWNtdhG0vqNOmCc6Ym08u8hLyaVn2nKwPFA/640?wx_fmt=jpeg)

近日，神策数据签署 CLA(Contributor License Agreement，贡献者许可协议)，正式加入 openEuler 社区。

神策数据（Sensors Data），全称神策网络科技（北京）有限公司，是国内专业的大数据分析和营销科技服务提供商，为企业提供神策营销云、神策分析云、神策数据根基平台三大产品方案，通过全渠道的数据采集与全域用户 ID 打通，全场景多维度数据分析，全通道的精准用户触达，帮助企业实现数字化经营。

神策数据未来将积极参与 openEuler 社区合作，与行业内的生态伙伴共建平台，推动行业生态繁荣发展。

同时，神策数据的大数据和营销科技产品（神策分析、标签画像及营销科技产品）获得华为鲲鹏兼容性认证。

神策数据联合华为计算开源 Arm 生态团队、北京鲲鹏联合创新中心，基于华为已在上游社区完成 Arm 使能的开源软件版本，经过功能、性能、安全等多轮测试，标志着神策大数据和营销科技产品（神策分析、标签画像及营销科技产品）已与华为鲲鹏完成兼容性互认。该方案基于鲲鹏全栈创新，使用 openEuler 20.03 LTS 操作系统，使用神策大数据和营销科技产品，可以整合广告投放、用户行为、业务经营等多种数据源，覆盖全场景的业务分析与用户（群）洞察，为企业中的不同角色提供实时、多维度的数据分析和智能决策方案。

来自神策数据的首席架构师、Apache HBase PMC 主席张铎，对于本次认证表示：“华为在开源上游及 openEuler 社区中的工作确保了在本次认证的顺畅进行，神策数据的大数据和营销科技产品涉及的多款开源软件，例如 Hadoop、Kafka、Zookeeper、MariaDB 等，均直接取自上游社区，并且在鲲鹏架构下直接可用。而对于 Kudu、Impala 等暂未使用到社区最新版本的软件，也可以通过简单的回合 patch 的流程，构建可用版本，极大地便利了下游厂商的集成及后续调优。

openEuler 操作系统及社区项目 BiShengJDK 等在 Arm 架构上所做的优化对于神策数据业务在多样性算力平台上的高效运行也提供了很好的支持，也为业内生态伙伴及开发者提供了很好的交流平台，期望后续能够与 openEuler 社区有更多的合作”
