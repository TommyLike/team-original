# [RISC-V 处理器开发商优矽科技加入 openEuler 社区](https://mp.weixin.qq.com/s/ADMkoBOPLdxbmrhH76OcoA)

[OpenAtom openEuler](javascript:void%280%29;)*2021-04-05 16:00:42*

近日，优矽科技签署 CLA，正式加入 openEuler 社区。

![](https://mmbiz.qpic.cn/mmbiz_jpg/A0h5yD51CMZdf8bdSrrXn1IwINdZbJOxibicYJDcVAu3eSOLnlbgnSWrp3ekAvyqL7ETazh4CdVCgDzEEicasmcZg/640?wx_fmt=jpeg)

优矽科技是一家基于 RISC-V 开源指令集提供硅知识产权产品设计及处理器开发服务的中国公司。2017年加入 RISC-V 国际基金会，务实聚焦于低功耗高性能处理器芯片领域的协同开发。

RISC-V（发音为“risk-five”）是一个基于精简指令集原则的开源指令及开放的处理器架构生态，简易解释为开源软件运动相对应的一种“开源硬件”。目前 openEuler 已经支持 RISC-V 架构，可以成功运行在中科院软件所的果壳处理器中。

今后，优矽科技将与中科院计算所携手，共同探索 RISC-V 架构下的开放单板计算机、科学计算库及开源软件开发工具包，探索 openEuler 在 RISC-V 架构处理器上的更多可能。
