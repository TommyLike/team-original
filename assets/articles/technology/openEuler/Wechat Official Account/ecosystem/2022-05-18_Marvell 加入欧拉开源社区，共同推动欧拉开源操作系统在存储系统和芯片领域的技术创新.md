# [Marvell 加入欧拉开源社区，共同推动欧拉开源操作系统在存储系统和芯片领域的技术创新](https://mp.weixin.qq.com/s/YuReEAicqHbVvVHKa1PZ6A)

[OpenAtom openEuler](javascript:void%280%29;)*2022-05-18 20:00:39*

近日，Marvell正式签署 CLA（Contributor License Agreement 贡献者许可协议），加入欧拉开源社区。Marvell加入欧拉开源社区后，将极大地繁荣多样性计算生态，推动欧拉开源操作系统在存储系统及存储芯片相关方向上的技术创新。

![](https://mmbiz.qpic.cn/mmbiz_jpg/A0h5yD51CMbFoyD5L9sQqFTHy2DW8taJa5o5omEKhxh1o6qrrhEHadUXJDyXJ1VvB4lLDp9GxdjeeCrbe6nanA/640?wx_fmt=jpeg)

Marvell 是基础设施半导体解决方案的全球领导厂商之一，凭借领先的知识产权和深厚的系统级知识，Marvell 基础设施半导体解决方案不断为企业、云、汽车、工业和消费类市场带来变革。今天，突破性创新仍然是Marvell存储、处理、网络、安全和连接解决方案的核心。

Marvell 作为全球领先的半导体解决方案厂商，积极参与国际开源社区贡献，是 Linux Kernel 社区和 DPDK 项目中活跃的贡献者，在参与开源社区建设方面有丰富的经验，为欧拉开源社区走向国际化提供经验指导。

Marvell 深度参与欧拉开源社区兼容性SIG和存储SIG的工作，推动欧拉开源操作系统与更多存储硬件兼容适配。目前，Marvell 旗下 Fibre Channel控制器（QLE269x、QLE274x、 QLE277x） 、NVMe控制器、SATA/SAS控制器等均已适配 openEuler 20.03 LTS SP3 和 openEuler 22.03 LTS。

欧拉开源社区欢迎全球更多芯片、板卡、整机厂商、ISV、开发者加入，共建面向数字基础设施的开源操作系统，共创最具创新力的开源社区。
