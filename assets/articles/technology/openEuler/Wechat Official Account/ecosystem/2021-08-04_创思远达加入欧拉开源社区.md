# [创思远达加入欧拉开源社区](https://mp.weixin.qq.com/s/wuFMpIetF7wOvoVla-mHaA)

[OpenAtom openEuler](javascript:void%280%29;)*2021-08-04 19:00:00*

近日，中科创达子公司创思远达签署了贡献者许可协议（Contributor License Agreement, CLA），正式加入欧拉开源社区。

作为一家以操作系统为核心技术的企业，创思远达将借助现有优势，积极参与欧拉开源社区合作，与业内的生态伙伴携手共建创新平台，推动行业生态繁荣发展。

在欧拉开源社区近期举办的 TC 会议上，来自创思远达的开发者申请的 OpenBoard 和 Edge 两个 SIG 也顺利通过。

其中，OpenBoard SIG 将通过定义开发板规范，推进欧拉开源社区在开放板卡上的软件生态发展，同时该 SIG 通过和 Open Board Task Force 合作，共同推进基于 RISC-V 的软硬件的标准化，促进 RISC-V 社区发展。

而 Edge SIG 的成立，主要是充分利用创思远达在边缘计算方面的优势，以欧拉操作系统为底座，构建基于边缘计算的软件仓库，和合作伙伴们共同定义标准的边缘计算平台。

未来，创思远达将基于自身领先的操作系统技术，与欧拉操作系统进行深度融合。如：以欧拉操作系统为底座，创思远达可聚焦更多领域业务；利用创思远达在边缘计算、人工智能等方面的深度积累，可对欧拉操作系统提供充足的技术支持，更有利于完善未来软件生态。

此外，创思远达对客户需求的深度了解，可为客户提供端到端的解决方案，能更有效的夯实欧拉操作系统在客户侧的表现，推动欧拉操作系统的产品化和产业化落地。

最后，创思远达作为欧拉开源社区的一份子，期待能和社区以及伙伴们共同推动社区的繁荣和发展。

#### 关于创思远达 Ainergy

中科创达旗下北京创思远达科技有限公司，致力于为物联网厂商提供端到端全栈解决方案和技术服务。创思远达凭借在物联网、边缘计算、人工智能和 5G 等领域的领先技术，与产业链各方的紧密合作，致力于在智能物联网技术上不断投入开发，为客户和合作伙伴提供高品质的产品技术服务。
