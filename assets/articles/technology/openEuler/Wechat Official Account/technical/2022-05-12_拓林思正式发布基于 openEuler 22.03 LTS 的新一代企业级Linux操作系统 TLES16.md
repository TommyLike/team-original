# [拓林思正式发布基于 openEuler 22.03 LTS 的新一代企业级Linux操作系统 TLES16](https://mp.weixin.qq.com/s/8qwgAEJMdOzQubjAwvJ98A)

*拓林思*[OpenAtom openEuler](javascript:void%280%29;)*2022-05-12 19:30:00*

近日，拓林思正式发布基于 openEuler 22.03 LTS 的企业级 Linux  操作系统，TurboLinux Enterprise Server 16（简称“TLES16”）。

TLES16是面向云环境和企业IT的强大企业级Linux系统，具备**高效、稳定、安全**等产品特性，致力于为企业级的数据库、大数据、云计算、人工智能平台提供安全稳定、7\*24小时不间断运行的技术支撑。

用户可基于TLES16，在**裸机、虚拟机、容器**及**云环境**中扩展现有应用并部署各类新兴技术，轻松驾驭各种环境并支持各项工作负载。

**系统特性**

**TLES16 的特性包括：**

- 基于首个全场景长周期版本 openEuler 22.03 LTS。
- Linux Kernel 5.10，集成社区最新稳定版本补丁、上游最新的性能增强，以及 openEuler 内核的各类优化。
- 支持最新的硬件平台：Intel Sapphire Rapids、华为鲲鹏920、飞腾S2500，内置大量常见硬件的最新驱动。
- 支持软实时操作系统内核。
- 轻量级性能图形监测工具。
- 内存分级扩展 etMem。
- 最新的虚拟化相关组件：QEMU和 libvirt，集成了社区的大量增强。
- 最新的容器相关组件：Docker，runc，containerd ，iSulad等。

**TLES16**的发布是拓林思坚持“繁荣生态 社区共建”的进一步举措。拓林思自2020年加入欧拉开源社区、成为**社区首批理事会成员单位**以来，为社区开源操作系统做了大量的代码贡献，分别合并到**openEuler** **20.03 LTS、openEuler 22.03 LTS、openEuler 20.09、openEuler 21.03、openEuler 21.09**等社区版中，并基于openEuler 20.03 LTS、openEuler 22.03 LTS分别发布了**拓林思商业发行版TLES15和TLES16**。

此外，拓林思参与了GNOME、XFCE、cinnamon、Desktop、Base-service等SIG组的开发工作，尤其在**GNOME、XFCE** 2个SIG组中，拓林思技术团队贡献占比**超90%**。

在欧拉开源社区会员贡献度排名看板中，拓林思贡献排名始终**位列前8**，尤其近1个月贡献排名中，拓林思**位列第4**，企业排名上仅次于华为和麒麟软件。

未来，拓林思将继续依托企业自身业务和技术优势，**持续强化与欧拉开源社区的合作共享，全力参与欧拉开源生态建设**，为社区成长和生态繁荣贡献力量，为操作系统产业发展添砖加瓦，为数字经济发展保驾护航。

**关于拓林思**

北京拓林思软件有限公司（TurboLinux,以下简称拓林思）成立于1999年，是国内第一家专业从事Linux开源操作系统开发和技术服务的软件公司，是经政府认定的国家高新技术企业、软件企业，并通过了ISO9001国际质量体系认证。

拓林思是公认的Linux中文化方面的奠基者，社区国际化118N(中文支持）代码的贡献者。1999年3月发布业内第一个Linux中文版TurboLinux 3.0.2中文版，开创Linux中文化先河。2001年9月TurboLinux简体中文版成功通过国家GB18030标准的符合性测试，成为第一个符合GB18030标准并通过测试的Linux系统。

拓林思是国内Linux企业级服务器市场的拓荒者，在国内Linux企业级服务器市场2001至2005年市场占有率第一，2000年起公司专注于推广Linux及开源软件在企业级的应用，提供高性能、高可用性的基于Linux和开源软件的企业级解决方案和服务，拥有众多成功案例和经验，客户遍布金融、电信、能源、交通、邮政、互联网、流通、政府、教育等各行业。且承担了中石化IC卡加油工程、中国工商银行Linux前置机系统、铁道部铁路行包营运管理系统、上海超级计算中心10万亿次高性能计算系统、国家质量监督检验检疫总局网络及三电工程、中国移动一级经营分析系统等国家大型重点信息化建设项目。

2006年，拓林思控股股东万里数据库和MySQL合作成立MySQL中国研发中心，在业务重心转向数据库的同时，继续为大型企业级客户提供优质服务如服务中石化二十年。在当前国际形势下，国内Linux操作系统市场迎来了新机遇，拓林思将发挥二十多年来企业级Linux市场的丰富积累，助力国内基础软件建设。
