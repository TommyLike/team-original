# [中科红旗携手华为、openEuler 社区、openGauss 社区，推出数字化转型方案](https://mp.weixin.qq.com/s/ozlmKDuohSCTFWHp0-PH4w)

![](https://mmbiz.qpic.cn/sz_mmbiz_png/1I4wicz1LrcWiaBsWvMoZ8JjySudWATVftiaZJWOECbYZSrpJhwZJGhXuPDj6PxF0YmBdpHARtSbzy48EXjukqSVg/640?wx_fmt=png)

随着“云+AI+5G”时代的高速到来，促使整个社会加速从物与物的连接、人与人的联接转化到万物互联，在这一过程中，不仅需要面对随之而来的海量数据和各式全新业务形态，更要满足对多元算力、多元场景以及网络和数据安全等全方面的综合考量与要求。

出于对未来行业与产业发展方向的高度认同，近日，中科红旗（北京）信息科技有限公司携手华为、鲲鹏社区、openEuler社区以及openGauss社区，推出基于鲲鹏处理+红旗Aisanux服务器操作系统（欧拉版）+openGauss数据库的软硬一体化数字解决方案，将中科红旗和华为以及知名三大技术社区的软硬件技术有机地、系统地、全面地整合在一起，为企业级客户和场景需求提供一体化的数字化转型解决方案。

**鲲鹏官网（https://www.hikunpeng.com）**

鲲鹏处理器是华为在2019年1月向业界发布的高性能数据中心处理器。目的在于满足数据中心的多样性计算和绿色计算需求 ，具有高性能，高带宽，高集成度，高效能四大特点。产品系列包括低功耗级鲲鹏916处理器：采用16nm工艺，支持24个内核，主频2.4GHz，功耗低至75w ; 极致效能级鲲鹏920-3226和鲲鹏920-4826处理器：采用7nm工艺，支持32和48个内核，主频2.6GHz，单位功耗的计算性能表现优秀 ; 极致性能级鲲鹏920-6426处理器：采用7nm工艺，支持64个内核，主频2.6GHz。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/1I4wicz1LrcWiaBsWvMoZ8JjySudWATVftzrTPkmIufIOcz8wwQNVEW1ySCGQ2oBl0KjT6SPdjVGExnQuFFaq7nA/640?wx_fmt=png)

**红旗官网（https://www.chinaredflag.cn）**

红旗Asianux服务器操作系统是中科红旗面向企业级客户推出的高性能服务器操作系统，在与华为和openEluer社区的合作基础上，相继推出了红旗Asianux服务器操作系统（鲲鹏版）V7.5、红旗Asianux服务器操作系统（欧拉版）V8.0等产品系列。其中红旗Asianux服务器操作系统（欧拉版）是基于openEuler20.03 LTS SP1进行二次开发的商业发行版，支持AArch64、X86\_64及其它多种处理器架构，能够充分释放计算芯片的潜能，适用于数据库、大数据、云计算、人工智能等应用场景。历经长久的开发调试与不断的优化下，在RedFlag Asianux Server - openEuler 8.1上成功实现了openGauss的快速安装与平稳运行且深度融合，体验良好。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/1I4wicz1LrcW7wRg0sFE78uu3tYoFxv2J9ul6DzMpfiaqV4FTV8r09tfmduapb6MjsBS8iboFtfw2O5fZ5cOlibEQA/640?wx_fmt=png)

**openGauss官网(https://opengauss.org)**

openGauss是一款开源关系型数据库管理系统，采用木兰宽松许可证v2发行。openGauss内核深度融合华为在数据库领域多年的经验，结合企业级场景需求，持续构建竞争力特性。同时openGauss也是一个开源的数据库平台，鼓励社区贡献、合作。
