# [统信服务器操作系统 V20 欧拉版发布](https://mp.weixin.qq.com/s/4_qE7b0bIwaGTxvIuZudaA)

[OpenAtom openEuler](javascript:void%280%29;)*2020-07-15 19:18:26*

转载自：统信软件

原文链接：

***https://mp.weixin.qq.com/s/P9DnIqMf-JP\_oPu17DMehw***

今日，统信服务器操作系统V20欧拉版（Uniontech OS Server 20 Euler）（简称：统信服务器欧拉版）正式对外发布。

![](https://mmbiz.qpic.cn/mmbiz_jpg/WfWJ2z36w1D7J2XhXuWhClf0o1EKzrNfTkHKGGyeicibicZvHvLFMjvJShA15KtqEEQy5Ta7ibplDNdUqKQRicvWJUg/640?wx_fmt=jpeg)

统信服务器欧拉版是基于openEuler 20.03 LTS社区发行版进行二次商业化发行的Linux操作系统，它采用Linux kernel 4.19、GCC 7.3、Glibc2.28等组件作为系统运行的基础环境，整合了自主研发的桌面运行环境DDE和常用的图形界面应用程序，适配了大量的第三方开源、商业闭源等软件和解决方案。

![](https://mmbiz.qpic.cn/mmbiz_jpg/WfWJ2z36w1D7J2XhXuWhClf0o1EKzrNfC6TTajMX7zlkDlNgXciaQp1hrHUn07rstPbrXe8buibL44lVcZxR1rmA/640?wx_fmt=jpeg)

统信服务器欧拉版针对华为泰山服务器鲲鹏系列处理器的稳定性和运行性能参数，进行了持续测试和深度调优，力臻充分释放鲲鹏处理器的一切潜能。

统信服务器欧拉版产品主要面向云、虚拟化、容器、大数据、人工智能等应用场景。支持用户根据场景需求自定义安装不同组件包，支持本地、网络方式安装，提供ISO、容器、云镜像交付物进行环境部署。

![](https://mmbiz.qpic.cn/mmbiz_png/WfWJ2z36w1D7J2XhXuWhClf0o1EKzrNfB8ydZgZZ8y1GMAPyuX0r52EKlyqD6K8l49jKbBPBWAKG7hAicyiczCkg/640?wx_fmt=png)

统信服务器欧拉版适配了统信软件旗下最新版DDE桌面环境，可以为用户提供良好的人机交互功能，符合国人操作习惯。此外，统信服务器欧拉版在安全性、稳定性、易用性方面具有较强优势，适用于电子政务、金融、电力、军工、交通等关键行业领域。

随着统信服务器欧拉版的发布，统信软件将携手华为共同为客户提供安全、稳定的操作系统产品与服务。未来，统信软件也将持续支持和深入参与openEuler社区生态建设。
