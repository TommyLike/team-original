# [云原生赋能博云企业转型，openEuler加速自主创新](https://mp.weixin.qq.com/s/Vsk3Jfo7jGXsjYZ9z-YCng)

[OpenAtom openEuler](javascript:void%280%29;)*2025-02-20 21:06:00广东*

在当今数字化转型的汹涌浪潮中，云计算正经历着从传统服务器时代向云原生时代的伟大跨越，已然成为驱动各行业变革的核心力量。近日，博云科技联合创始人郭伟做客人民网巨人港会客厅，就云原生技术的发展现状、博云在该领域的实践成果，以及这项技术对企业和大众生活产生的深远影响，进行了深入且全面的解读。

云原生技术，作为云计算时代的产物，是云环境下软件开发、部署与运维的全新方式。博云早在 2014 年便敏锐地捕捉到了这一趋势，毅然投身云原生市场，专注于云原生技术软件与 AI infra 技术软件领域。郭伟指出，对于云原生企业而言，保持竞争力的关键在于持续不断的研发投入。博云在这方面一直不遗余力，研发投入费用占营收比例高达 20% 以上。通过技术创新，博云做到了 “专” 与 “通” 的完美结合。同时，博云积极拥抱开源，通过同 OpenAtom openEuler（简称“openEuler”）社区紧密合作，基于开源技术，极大缩短了在云原生等基础设施领域的投资回报周期，提升了博云业务自主创新的速度。

**博云与 openEuler 深度合作**

openEuler 作为国内开源操作系统社区的领导者，与博云的发展策略不谋而合。openEuler 从 22 年就提出了全面支持包括云计算场景在内的全场景的技术目标，并先后在 openEuler 社区推出多个重量级云原生相关项目，如 iSula、Kmesh、KubeOS、StratoVirt 等。

与 openEuler 的合作进程里，博云积极承担生态建设责任，成为 openEuler 白银捐赠人，以实际行动助力开源社区的蓬勃发展，推动 openEuler 技术的持续演进与广泛应用。

博云在开放原子开源生态大会上发布了云原生 + openEuler + 鲲鹏联合解决方案。该方案深度融合了博云在云原生技术领域的深厚积累以及 openEuler 操作系统的开源优势，从系统架构优化、性能调优到安全防护机制的构建，都进行了全方位的精心打磨。在系统架构上，通过微服务架构设计，实现了各功能模块的高效协同，提升了系统的灵活性与可扩展性；性能调优方面，基于 openEuler 的内核特性，优化了资源调度算法，大幅提升了系统的响应速度和吞吐量；安全防护上，引入了多重加密技术和实时监控机制，有效抵御各类安全威胁 。这一联合解决方案，能够为企业客户提供更加高效、稳定且适配 openEuler 操作系统的专业方案，进一步拓宽了 openEuler 在各行业的应用边界，提升了其市场竞争力。

在技术实力的竞技舞台上，博云同样表现出色。积极参与鲲鹏大赛，凭借基于 openEuler 精心打造的创新方案，从众多参赛团队中脱颖而出并获奖，充分彰显了博云在云原生技术与 openEuler 结合应用方面的深厚技术积累与创新能力。方案基于 openEuler、isulad 等关键技术，在容器安全性方面通过深度安全防护机制、安全隔离技术以及漏洞管理与修复等措施，实现了全方位的保障；在并发性能上，凭借高效的资源调度算法、轻量化容器运行时和并发连接优化，极大提升了系统的响应速度和处理能力，准适配资源体量大且对资源使用率敏感的客户群体，为企业数字化转型提供了坚实可靠的技术支撑。

郭伟还透露，博云已经与昇腾合作推出了 AI 模型一体机，并且未来将继续以打造自主创新云原生底座为核心要务，与 openGuass 数据库等进行适配和性能调优，全方位赋能企业数字化转型。

谈及未来五到十年的数字生活，郭伟认为，数字化和智能化将更加深入大众的生活和工作。目前，AI 工具已经在提升工作效率方面发挥了重要作用，未来这种趋势将愈发显著，甚至可能出现 AI 机器人与人交流的场景。openEuler 一直以来坚持从 OS for AI 和 AI for OS 的角度持续推进对于 AI 的支持。openEuler 推出的 openEuler Copilot System 大模型人机交互界面，不仅支持 OS 本身的智能交互运维，还提供智能体的开发框架，帮助客户开发面向自身业务的智能体。云原生作为未来 AI 业务的基座，博云将和 openEuler 在 AI 的云原生开发部署环境一键获取、开箱即优、训推加速提效等方面进行更深入的合作。云原生技术不仅在重塑企业 IT 架构，推动企业数字化转型方面发挥着关键作用，也在悄然改变着大众的生活方式。

未来，博云和 openEuler 将继续秉持创新精神，不断深化合作，在云原生技术领域持续深耕，为推动企业数字化转型和改善大众数字生活贡献更多的力量。

相信在博云和 openEuler 的不懈努力下，云原生技术将绽放出更加耀眼的光芒，为各行业的发展注入新的活力，引领我们走向更加美好的数字未来。
