# [openEuler 技术委员会委员熊伟当选 2020 中国开源杰出贡献人物](https://mp.weixin.qq.com/s/8_znxkvjsujCxwVQXVVaKQ)

*周主编*[OpenAtom openEuler](javascript:void%280%29;)*2020-07-20 20:30:00*

2020年7月17日，在第十五届“开源中国开源世界”高峰论坛中，openEuler 技术委员会委员熊伟，当选 2020 中国开源杰出贡献人物。

![](https://mmbiz.qpic.cn/mmbiz_png/A0h5yD51CMZxWwbiaNJujsUmqy8sst8I6mbwiaw1FAI3oDPdt8lfibnL6RRarCfK0Q38ibsoSNibaWntkDHLMibAicnTQ/640?wx_fmt=png)

熊伟毕业于南开大学，获工学博士学位。他是操作系统领域的一个老兵了，分别在TurboLinux，WindRiver 等公司参与 Linux 操作系统的研发。2014年加入华为，现为 2012 实验室中央软件院服务器操作系统首席架构师，openEuler技术委员会委员。也是 openEuler 社区的主要创建人之一。

作为社区的发起人之一，熊伟博士对 openEuler 社区寄予厚望，希望能够和行业内的同仁们一起构建一个支持多算力，多体系架构的 Linux 开发平台，通过开放的社区形式与全球的开发者共同构建一个开放，创新、多元的软件生态体系。熊伟博士对 openEuler 社区的定位是从源头做起，成为能够独立演进的一个操作系统开放体系，不但能够为厂商提供稳定可靠的操作系统底座，也能将 openEuler 打造成为一个创新的舞台，帮助业界孵化新的想法，新的技术，新的实现。

熊伟博士表示，“十分荣幸能够当选 2020 中国开源杰出贡献人物，这不仅仅对我个人，更是对以openEuler为代表的中国蓬勃发展的社区的鼓励。感谢大家对我和openEuler 社区的支持，希望在未来的日子里，openEuler能集合更多的开发者，爱好者，成为大家爱用，好用的顶级开源社区。”

在此，openEuler 社区再次祝贺熊伟博士荣获2020开源杰出贡献人物，也希望openEuler 社区能够涌现出更多的开源杰出人物，为中国的开源事业贡献我们的一份力量。
