# [openEuler + CULinux打造联通云新一代基础设施](https://mp.weixin.qq.com/s/Hk_qYckh1uTY9sriHDfgQA)

[OpenAtom openEuler](javascript:void%280%29;)*2022-04-20 08:55:09*

在刚刚落下帷幕的欧拉开发者大会2022（openEuler Developer Day）上，欧拉首个数字基础设施全场景长周期版本openEuler 22.03 LTS正式发布。这是欧拉捐赠之后的首个社区共建版本，来自社区的 773 名贡献者、百家厂商和用户参与了版本开发，新增代码2300万行。该版本采用统一的 Linux Kernel 5.10 内核，支持服务器、云计算、边缘计算、嵌入式等应用场景。

联通云CTO钟忻受邀出席大会，发表《openEuler+CULinux打造联通云新一代基础设施》的主题演讲。他在发言中表示联通云一直坚持开源与商业结合的技术路线来发展云计算基础架构技术体系，并介绍了联通与欧拉的合作历程，以及联合创新成果。

联通云深入基础架构组件改造，联合openEuler进行操作系统内核优化，加强与异构和硬件计算能力的融合，基于虚拟化和云原生的双引擎技术，融合七大场景创新，包括物联感知云、数海存储云、智能视频云、智链协同云、5G边缘云、自主创新云、混合云，发挥联通云“云边端网管业”综合优势，推动各领域数字化优化升级，全方位加速政企客户数字化变革。

2020年7月联通沃云加入openEuler社区，成为OpenStack SIG首批maintainer及贡献者，深度参与openEuler社区开源贡献及SIG 组日常运作，主导将OpenStack V版移植到openEuler 21.03版本，使openEuler初步实现云化基座。随后持续参与OpenStack在openEuler操作系统的适配工作，先后将R、W、T等版本移植到openEuler 20.03 LTS、openEuler 21.09等版本。联通也积极参与了openEuler 22.03 LTS版本贡献，完成openEuler支持多版本OpenStack适配工作。

在openEuler Summit 2021上，联通推出了首个自主研发设计的操作系统——CULinux数据中心欧拉版。这是联通在核心技术领域向前迈进的一个重要里程碑，为联通云的基础设施带来一次焕新升级，从运行环境上为联通云提供可靠的基础。

随着欧拉社区新版本的发布， CULinux结合openEuler 22.03 LTS，借助Linux Kernel 5.10内核能力，支持异构多样性算力平台，并充分结合中国联通轻量级虚拟化、高性能网络、安全容器等技术，为联通云基座、云原生、边缘站点等多层次的技术架构提供稳定支持。未来，该版本将在联通云全场景内展开部署。

CULinux在六大方向进行了优化：

1. CULinux结合openEuler，依靠Linux Kernel 5.10内核能力，具备内核热升级技术；
2. 集成了轻量级虚拟化组件，相比传统虚拟化提升了 10 倍的启动速度、并且大幅降低资源损耗;
3. 多种加密措施为数据及运行环境的安全防护提供稳定保障;
4. 支持智能化内核参数调优，可更好地为大并发网络、高 IO存储提供支撑;
5. 集成部署工具与监控组件实现高效运维；
6. 广泛适配兼容国产化软、硬件生态，为信创产品运行保驾护航；

目前，已经面向服务器、边缘计算、云基础设施等多种场景，提供多个定制版本。未来，中国联通将继续研发更多的 CULinux 发行版本，覆盖更广泛的应用场景。

在发言结尾，钟忻表示：“联通坚持走开放合作的发展之路，与上游供应链厂商、业务应用厂商和客户加强产业生态合作，与openEuler等开源社区一起开辟新技术领域，积极促进开源+商业的正循环。”

联通作为欧拉社区理事成员，将继续发挥在云基础设施领域的能力，投入openEuler云生态的工作中，并将加大对创新研发方面的贡献，围绕容器化部署支持、云场景平滑升级、边缘计算等方面进行重点开发。期待更多的开发者和企业参与openEuler社区，共建数字基础设施的开源操作系统。
