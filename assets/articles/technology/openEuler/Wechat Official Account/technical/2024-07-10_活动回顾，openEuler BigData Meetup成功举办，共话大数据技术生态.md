# [活动回顾，openEuler BigData Meetup成功举办，共话大数据技术生态](https://mp.weixin.qq.com/s/JhE-Je5XRbQxZTLbcYzvDQ)

[OpenAtom openEuler](javascript:void%280%29;)*2024-07-10 18:15:52中国香港*

2024年7月5日， OpenAtom openEuler（简称"openEuler"） BigData SIG 举办了一场线上Meetup活动。此次 Meetup 汇聚了大数据领域的技术专家和行业先锋，下面就让我们来回顾本次 Meetup的精彩内容吧~（文末有彩蛋）

下面就让我们来回顾本次 Meetup 的内容

Arm China顾煜祺在议题“ Empower BigData on Arm64 ”里详细分享了大数据项目生态在Arm平台上的应用和性能情况，介绍了在Apache项目：Apache Bigtop和Apache Arrow上的应用和优化工作。老师还和我们一起探讨了Spark SQL加速方案：Gluten和Velox，并分享Velox在Arm平台上的一个优化案例和一些针对大数据工作负载的Arm性能分析工具。

Senior Tech Lead，Linaro Ltd.徐国栋在议题“ UADK加速大数据：鲲鹏服务器的关键应用场景 ”里详细分享了“UADK加速大数据”项目，这是华为鲲鹏服务器的关键应用之一。探讨了在大数据领域使用UADK时的集成挑战，并分享Linaro落地团队在大数据全栈软件模块集成方面的成果，还详细讨论UADK在加速HDFS Transparent Encryption在鲲鹏上的性能优势，以及实测的TeraSort/HDFS透明加密（SM4-CTR）的处理速度提升。

Bigtop Manager是一款现代化的，面向云原生的大数据集群管理平台，提供了一套轻量级的方案高效地管理bigtop所支持的大数据组件。经过在openEuler社区Bigdata SIG孵化，6月份毕业进入Apache社区。openEuler Bigdata SIG Committer 吴治国在议题“ Bigtop Manager：面向云原生的大数据集群管理平台 ”里分享了Bigtop Manager的立项背景，当前进度，架构设计以及未来规划等。

openEuler Bigdata SIG Maintainer陈强在议题“ 一种高效的Spark SQL Native引擎实现方案 ”里分享了一种C++实现的Spark SQL Native引擎解决方案，基于Spark提供的Extension功能插件式实现无需侵入修改代码。动态代码生成和静态编译结合并使用LLVM动态优化生成代码，算子自动向量化，提升Spark SQL的性能。
