# [为什么你应该看一看华为系统软件创新峰会](https://mp.weixin.qq.com/s/9ErWLV26fVc8Y-2OW0zMYQ)

[OpenAtom openEuler](javascript:void%280%29;)*2021-03-27 16:54:30*

编辑推荐：

华为作为 openEuler 社区的创始会员，同时也作为 openEuler 社区中内核贡献的主要团队，在 Linux Kernel 社区上持续推动内核技术的发展与创新。

本次华为系统软件创新峰会围绕 Linux 操作系统这一主题展开，来自华为中央软件研究院、华为欧洲研究院的技术专家、 Linux Kernel 社区中的 Maintainer 和其他业界专家一起，来聊一聊系统软件创新这个话题。

欢迎 openEuler 社区中的开发者报名参与本次英文线上峰会。
