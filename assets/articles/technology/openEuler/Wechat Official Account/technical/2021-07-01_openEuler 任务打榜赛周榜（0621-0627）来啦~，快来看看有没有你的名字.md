#### 关于鲲鹏应用创新大赛·openEuler 赛道

鲲鹏应用创新大赛·openEuler 赛道是面向 openEuler 社区开发者的个人赛。

开发者可以通过参加大赛，参与 openEuler 社区最新特性开发，掌握操作系统领域的最新技术方向。

赶快来 openEuler 社区参与代码贡献吧，让我们携手共建最具活力的操作系统开源社区。

openEuler 赛道又分为**任务打榜赛**和**特性命题赛**。

#### 关于 openEuler 任务打榜赛

针对社区内的小任务，从小入手，上手简单，开发者直接参与社区贡献，为社区创造价值。

每天下午 2:30 准时发布三十道赛题，每道赛题合入即可获得相应的积分。

在比赛中，一些开发者提交的 PR 得到了社区 Maintainer 的认可与表扬，同时我们找邀请到5月月榜的冠军 yaozc07，[对他进行了一次采访](http://mp.weixin.qq.com/s?__biz=MzI2NDE4OTE2Mg%3D%3D&mid=2247488841&idx=1&sn=a1e51a82d595afad99f6c395e9f12adc&chksm=eab130ccddc6b9da41f34e376c7a4c390b799e53937507e0f2f54f6e15ee42e4bec381a215b8&scene=21#wechat_redirect)，请他来谈一谈参与 openEuler 任务打榜赛的心得体会，相信他的分享可以帮到正面屏幕前看这篇文章想要参赛的你。

#### 关于 openEuler 特性命题赛

用于参与社区的中长期任务，明确目标，主要用于一些工具的开发，开发者可以深度参与社区贡献。

通过引入 SIG 组的导师，开发者直接与赛题的导师建立联系，与社区技术大咖直接交流学习。

openEuler 特性命题赛总计有8道赛题，其中区域赛有4道，半决赛有4道。半决赛的4道赛题如下：

- git log 转 change log 工具
- 软件编译时长优化
- python-fuzz 数据类型（数字/字符串/列表/字典）的变异支持
- MySQL 性能编译优化

半决赛的赛题将会在区域赛结束之后公布。
