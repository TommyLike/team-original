# [openEuler Kernel 技术分享 ｜ Arm 架构学习资料分享](https://mp.weixin.qq.com/s/YAOQmiXIO65VXcZV00OJ9Q)

原创*Kernel SIG*[OpenAtom openEuler](javascript:void%280%29;)*2021-05-12 18:00:00*

openEuler Kernel SIG 会持续规划一些技术议题，欢迎正在阅读的你一起加入，分享与探讨内核技术。

ARM 官网上的 Learn the Architecture 章节是非常好的 ARM 架构入门学习材料，整体介绍了 ARM 架构中 A/R/M 三个 Profile 的架构特性。

本次讲座将以官网上 A Profile 为例，介绍其中 Exception Model/ISA/Memory Management/Memory-Model/Security 等章节，让听众对 A Profile 有整体的理解。

网站链接如下：https://developer.arm.com/architectures/learn-the-architecture/a-profile
