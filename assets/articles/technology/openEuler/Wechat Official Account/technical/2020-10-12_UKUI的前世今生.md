## 什么是 UKUI

用户界面(User Interface-，简称 UI，也称使用者界面)是指对软件的人机交互、操作逻辑、界面美观的整体设计。它是系统与用户之间进行交互和信息交换的媒介，实现信息的内部形式与人类可接受形式之间的转换。

UI 一词最早出现在 1980 年代，当时苹果电脑的麦金塔(Macintosh)使用 WIMP（视窗、图键、窗体，以及鼠标鼠标）将图形用户界面(GUI)带进了大众市场，从而取代早期计算机使用的命令行界面。

UKUI(Ubuntu Kylin User Interface-)是由麒麟团队开发的基于 Linux 发行版的轻量级桌面环境，默认搭载在优麒麟开源操作系统和银河麒麟/中标麒麟商业发行版中，同时也支持 Ubuntu、Debian、Arch、openEuler、Fedora 等主流的 Linux 发行版。

## UKUI 的发展历程

2013 年 4 月，优麒麟开源操作系统经过 Ubuntu 技术委员会的严格评审，正式成为 Ubuntu 官方认可的衍生版，并同期发布第一个版本--优麒麟 13.04，其默认搭载的桌面环境为 UKUI 1.0 版本。

直至优麒麟 17.04 版本的正式发布，优麒麟推出了全新开发的 UKUI 2.0 桌面环境，UKUI 2.0 设计紧贴普通用户需求，特别是针对有一定 Windows 系统使用习惯的用户，减少其使用 Linux 系统的学习成本，获得了国内外爱好者的广泛关注，并被国际主流期刊报道。

2017 年，UKUI 桌面环境正式进入主流发行版 Debian 的官方仓库，成为第一款由中国团队主导开发并进入两大国际主流社区的桌面环境。

今年，UKUI 桌面环境和优客系列应用软件成功移植至 Arch Linux 操作系统。openEuler 社区技术委员会也审议通过，正式成立 UKUI 桌面环境特别兴趣小组（UKUI SIG）。目前 UKUI 已经成功支持 Ubuntu、Debian、Arch、openEuler、Fedora 等国际主流 Linux 发行版，未来将会有越来越多的 Linux 发行版将 UKUI 作为可选桌面环境。

## UKUI 的设计演化

七年前，优麒麟团队试图寻找一个友好、高效、美观并有中国特色的 Linux 桌面，于是 UKUI 就这样诞生了。相较于功能繁多与伴随而来的整体膨胀性，UKUI 更注重易用性和敏捷度，各元件相依性小，可以不倚赖其它套件而独自运行，并且能提供类 Windows 风格的交互功能，给用户带来更加亲切熟悉的使用体验。

UKUI 2.0 是一款基于 MATE 的轻量级桌面环境，使用 GTK 和 Qt 开发，从开始菜单、控制面板、桌面任务栏以及桌面主题图标等不同层面进行了重新设计和实现，内容丰富，界面清爽。在视觉上，UKUI2.0 遵循中华民族美学核心——观物取象，主色调采用了源自大自然的中国传统颜色，而图标及默认壁纸则是拟物化和扁平风结合统一的形式。在交互上，偏向 Windows 用户习惯，将默认任务栏改为了下方，并在文档管理器中引入树状结构，让用户能够更方便快速的适应系统使用，特别是那些已经熟悉 Windows 操作系统的普通用户，增加功能的实用性及易用性。

UKUI 2.0发展史

2019 年优麒麟团队启动全新的 UKUI 3.0 设计开发工作，无论从设计理念还是整体架构上，跟 UKUI 2.0 比起来都算是质的飞跃。UKUI 3.0 从视觉和交互上均遵循奥卡姆剃刀原理——如无必要，勿增实体（即“简单有效原理”），重构用户界面，化线为块，使用户更易聚焦内容区域；化繁为简，简化用户操作流程。全新绘制微渐变扁平风格图标，细节更加严谨，简约而不简单。增加暗夜主题、夜间模式等情感化设计，黑暗和光明会唤起不同的情绪，让系统不再只是冷冰冰的工具。

很早之前，听到过六个字“精于心，简于型”，它是从英文 Sense and Simplicity 翻译过来的，来源于荷兰皇家飞利浦电子公司的品牌承诺。顾名思义，“精于心”表达的是产品开发和创新必须从用户内心的需求出发来用心设计，“简于型”表明产品尽量趋于简单、智能、好用。这两点不光适应于工业设计，同样也适用于计算机软件及应用设计。原来的我们，只关注如何减少用户从 Windows 迁移过来的学习成本，而从 UKUI 3.0 开始才真正注重提升视觉效果，并致力做到一切以用户价值为依归——享我所想，懂你所向，因为需要，所以存在。我们也相信这个设计理念能让 UKUI 在未来的路上，越走越远。
