# [「转」中国OS崛起之路：活跃的openEuler开源社区是关键](https://mp.weixin.qq.com/s/kqr8fJzSxON8OGt4oQ6sVA)

*黄海峰*[OpenAtom openEuler](javascript:void%280%29;)*2020-12-30 22:42:25*

ARM、x86、RISC-V等多样性计算呼唤新的开源操作系统，支持跨平台、跨算力的需求。去年免费开源的openEuler应运而生，让中国多样性计算平台操作系统行业崛起。

**一年来，openEuler发展取得不少成绩**，吸引中国移动、中国联通、银联、飞腾等60多家单位加入。而麒麟软件、统信软件等操作系统厂商也发布了openEuler商用版本。

**开源社区有句名言：社区重于代码。**开源的openEuler要真正繁荣起来，获得持久发展，就需要繁荣社区。而组成社区的根基是千千万万个开发者，只有凝聚多方合力，才能共建开源生态。

“回到开源的本质和初心来看，我们需要传递的是开源精神以及文化的价值，在这样的精神指引下，我们才能更好地前行，以真正开放的心态合作和推进开源生态。”在最近召开的openEuler Summit 2020上，**开放原子基金会TOC（技术监督委员会）主席、华为云与计算开源业务总经理堵俊平如此表示。**

堵俊平是一位长期??的开源社区贡献者，他为何要强调**开源文化？**笔者疑惑，我们该如何看openEuler这些成绩，openEuler真的值得产业投入吗？openEuler社区要成为最活跃的开源社区，还要做什么？

**01**  **Linux改变世界：**

**openEuler承接新需求**

众所周知，硬件+操作系统是计算产业的基石，是核心技术。而从某种意义上讲，操作系统更为关键，因为所有计算硬件都需通过操作系统来调用，所有应用软件都需通过操作系统来使能，操作系统的竞争力直接影响信息产业的整体发展。

**在开源操作系统领域，国外发展领先，中国正在崛起。谈及开源，我们都会想到Linux。**

![](https://mmbiz.qpic.cn/mmbiz_png/gL0GByFsp5uvbhsCLqbS3GgP2ClicVoz8CpcbgsUfQFPqnLxObicVUBRNxW0Mm07bGjviaTLgQv7mZDibgicj4zBiaKQ/640?wx_fmt=png)

Linux从1991年出现，创造了一段段完美历史。比如2011年，谷歌推出 Chromebook；2014年，微软“示爱”Linux；2017年，所有的超级计算机都运行在Linux之上；2019年，微软为Windows10用户推出Linux。

可以看出，Linux改变了开发者的世界，也改变了商业世界的游戏规则：Linux出现打破了Unix一通天下的小型机历史，攻陷了Windows占领的PC桌面市场，到2017年所有的超级计算机都运行在Linux之上。

2019年12月31日，我们很欣喜地看到一股??新兴的中国开源力量诞生：基于Linux内核的openEuler正式开源。openEuler与以往的开源操作系统有何不同？

据介绍，未来云原生、智能边缘等新技术发展，为操作系统带来全新变化。传统操作系统面临多场景、新负载、分布式等诸多挑战，创新空间巨大。openEuler推出恰逢其时，openEuler将充分释放多样性算力潜能，是全场景协同的操作系统。

“openEuler可以为服务器等多种计算硬件提供稳定版的操作系统，补足市场空缺。”堵俊平介绍，业界希望openEuler可以基于开放社区，保持各方共同创新，满足和承接计算市场的新需求。

**02**  **成果喜人：**

**openEuler一年取得全方位发展**

**openEuler这一年犹如新生婴儿，成长速度惊人。**2019年9月18日，华为宣布将于2019年内开源服务器操作系统；12月31日，openEuler开源代码上线；2020年3月27日，openEuler 20.03 LTS版正式发布；9月30日，社区又发布了openEuler 20.09创新版。

值得一提的是，openEuler 开源社区在全球开发者的参与下，取得蓬勃发展，社区十分活跃，PR总和已经超过20000个。

**具体来看，第一**，openEuler社区已经有超过2000名贡献者、在全球120个城市近3万次下载、3万社区用户。**第二**，openEuler社区吸引了包括中国移动、中国联通、银联、飞腾等60多家企业、机构和高校的加入。**第三，**社区SIG特别兴趣组已增加到70多个，覆盖了内核、虚拟化、云原生、桌面等方向。

![](https://mmbiz.qpic.cn/mmbiz_png/gL0GByFsp5uvbhsCLqbS3GgP2ClicVoz8Hvn7RkzPRU5XWynTbxl8xWhD29DOOnHUzERCsjGpvrIoT6B6JdEOAw/640?wx_fmt=png)

openEuler summit 2020上颁发优秀贡献者奖

堵俊平表示，这些数字都非常出色，比如因为中国并没有那么多聚焦操作系统底层程序的开发者，从这个角度看，openEuler社区凝聚起的2000名贡献者数量就显得极为亮眼。

**笔者好奇，开源的openEuler是基于华为操作系统打造，为何有那么多开发者和企业机构愿意加入？经过与堵俊平交流，笔者总结出如下三大理由。**

**首先**，华为是Linux的主要贡献者。在最新的Linux Kernel 5.10版本，华为内核贡献排名已经升至全球第一。这意味着，华为能把十多年来在操作系统上积累的能力开放出来，可大幅提升整个openEuler操作系统产业的基础能力。

**其次**，华为坚持有所为、有所不为。华为自己不做操作系统的商业发行，全面使能合作伙伴借助开源的openEuler发展自有品牌的商业版本，快速实现商业成功。

据悉，现在中国主流的操作系统厂家麒麟软件、统信软件、中科院软件所、普华基础软件、麒麟信安、拓林思等，都已加入openEuler社区并发布了商业发行版，同时也在积极参与社区贡献。

**最后，**openEuler的确是大家共建的开源系统。许多企业、高校、研究机构等，积极参与openEuler社区贡献。其中，麒麟软件已经成为社区第二大贡献者，并在金融、电力等行业实现规模商用。

我们以openEuler 社区在9月30日发布的openEuler 20.09 创新版为例。该版本由openEuler社区中的多个公司、团队、独立开发者协同开发，比如中科院软件所贡献了RISC-V 新指令集架构支持，麒麟软件为社区共享了UKUI桌面环境。

这样的社区化协同大规模开发，在中国软件开发历史上当属首次，也充分体现了开放社区协同开发的优势。

**03**  **社区重于代码：**

**如何建最活跃的openEuler社区？**

开源社区是开源操作系统的关键。如何将openEuler社区，打造成中国乃至全球最活跃的开源社区？众所周知，一个开源社区需要传递代码、传递技术、传递商业成功，但这还不够。

堵俊平表示，吸引优秀的开发者们共同构建的社区，远远比代码重要。社区精神将召集到大量优秀的人，从而一起构建高质量的开源项目。“我们需要传递的是开源精神以及开源文化的价值。”

**发展openEuler，为何社区文化精神如此重要？堵俊平的观点并非凭空想象，而是有迹可循。**据悉，作为全球最大的开源基金会，Apache Way在迎来20周年庆祝时，持续坚持去宣扬的正是社区精神：Community over code 。翻译过来就是，社区大于技术，社区大于代码。

而这一点在CNCF基金会上也有突出体现。CNCF基金会是近年来发展最迅速，影响力增长最快的开源基金会，围绕着旗舰项目Kubernetes，在短短几年的时间里聚拢了30多个项目，构建起庞大的生态，吸引了无数的用户和开发者加入其中，掀起一场云原生的技术革命。

中国开源力量的崛起。今年在中国开源圈发生的一件大事是，中国成立了开放原子开源基金会。该基金会的口号是“ All for the Developer , All for the Universe” ，也强调文化精神。

**基于此，堵俊平分析，openEuler社区各方，可以在透明&分享、合作&共赢、创新&迭代、多样性&包容性等精神指导下，构建最活跃的开源社区。**

**第一，透明&分享。**堵俊平介绍，开源社区工作者会发现，相比在一个要求KPI的企业中工作的人，在社区里的许多开发者更能体会到合作的舒适性和安全性，也更容易建立起相互间的信任感。很重要的一个原因是开发者??彼此之间毫无防备，所有技术公开透明，可以公开讨论技术细节和决策。

**第二，合作&共赢。**在信任基础上，openEuler社区成员能够更好地合作并追求共赢，??开源天然就有这个合作的属性。“同一份代码既承载了你的使命，??也承载了我的诉求，所以我们可以一起??携起手来追求1+1&gt;2的价值，构建共赢社区”。

**第三，创新&迭代。**??在这样共赢的openEuler社区里，开发??贡献者包括软件??服务商，一起努力进行最广泛的讨论，不断迭代实现优质代码、优质的创新，成就各个软件领域创新的代名词。

**第四，多样性&包容性。**openEuler开源社区包容各种各样的人群进入社区，包容各种各样的想法进入社区。开源没有落下任何人，??没有落下任何伟大的idea。大家可以在社区来实现技术??迭代、创新创造，??任何人都可以来社区学习、交流，并贡献自己的专业和热情。

**04**  **共建、共享、共治：**

**全力推进openEuler社区建设**

据了解，未来3-5年，华为希望通过社区的共同努力，支持千家企业、万级开发者、百万级用户，把openEuler打造成最具活力的开源社区，构建起面向全球的主流操作系统生态。

**那么，如何持续推进openEuler社区建设？**

**一是**openEuler深入贯彻“共建、共享、共治”理念，广泛吸引合作伙伴和开发者共同参与openEuler开源社区贡献。

**二是**在研发投资上，持续保持在Linux开源社区以及openEuler内核的研发投入。如聚焦内核等关键技术创新，推出更有竞争力的openEuler社区版本。

**三是**openEuler社区在支撑中国操作系统产业百亿市场空间的同时，也会继续保持与全球开源体系融合，与各开源社区、开源项目广泛协作，积极推动全球开源软件技术生态更好地支持多样性算力。

据悉，openEuler为构建社区共建的制度保障，在本次峰会全面升级社区治理体系，正式**成立社区理事会、升级技术委员会**，技术、营销、人才培养等各项工作将完全按社区方式开放运作。

“问渠哪得清如许，为有源头活水来。”堵俊平在演讲中表示。??这个源头是源代码，华为??希望大家一起贡献??和维护好上游社区，吸引和培养优秀的人。

**记者观察：**

**积水成渊，蛟龙生焉**

据统计，我国信息产业营收中软、硬件比例为29：71，而全球平均水平为55：45。软件价值与市场价格的不匹配，无法长期可持续发展。

与此同时，各行业ARM、x86、RISC-V等多样性计算新需求显现，今年新基建等推动各行业数字化建设，加速各类计算平台落地发展。

这些都让我国操作系统产业迎来历史最好的发展机遇。不过，操作系统的竞争力打造和产业生态构建，是一个漫长而又艰辛的过程，需要战略耐心和持续投入。业界需要厚积薄发，开放合作，协同创新。

**独行快，众行远**，开放开源是软件技术创新的重要途径，充分利用开源、参与开源、支持开源，发展操作系统，联合做大做强是当前最为可行之路。

通过与堵俊平交流，笔者了解到，openEuler这个面向未来的开源操作系统，依托于极具活力的openEuler社区，发展迅速，且未来方向明确。它的未来或将不止于操作系统。

下一步我们需要做的是，重视基础软件能力，吸引更多生态企业和人才，基于openEuler社区一起开放透明发展开源操作系统。

积水成渊，相信中国操作系统的“蛟龙”终会腾空而起。
