> 开源软件供应链点亮计划-暑期2021（简称暑期2021），是由中国科学院软件研究所与openEuler社区共同举办的一项面向高校学生的暑期活动，旨在鼓励在校学生积极参与开源，促进优秀开源社区的蓬勃发展。主办方联合各大开源社区，针对重要开源软件的开发与维护提供项目，并向全球高校学生开放报名。学生自由选择项目，与社区导师沟通实现方案并撰写项目计划书。被选中的学生将在社区导师指导下，按计划完成开发工作，并将成果贡献给社区。根据项目的难易程度和完成情况，参与者将获得6000-12000不等的奖金。
> 
> - 活动官网 https://summer.iscas.ac.cn/

openEuler 社区在暑期2021活动中已上线100+项目任务（https://openeuler.org/zh/activities/），涉及操作系统内核、桌面环境、存储、安全、网络，以及与云计算、云原生、虚拟化、数据库、大数据、AI、Rust等多个方向的结合。

我们将组织一系列在线会议+直播，让社区技术大佬和大家面对面，介绍此次openEuler社区项目任务。

第一期将由openEuler Kernel SIG 带来内核方向的项目任务讲解，包含了内核调度、内存、文件系统、架构、调测多个模块等方向40+个任务，想参与内核开发的同学们不要错过哦！

Linux Kernel 是当前最主流的操作系统 Kernel， openEuler Kernel 衍生自 Linux kernel，并在可靠性、安全性、虚拟化、性能等方面做了针对性的增强，致力于打造最具活力的原生操作系统开源社区。

openEuler Kernel SIG 聚集了国内外各领域技术专家和优秀的 Linux Kernel 技术开发者，有浓厚的技术氛围。加入 Kernel SIG，你可以站在 Kernel 技术大牛的肩膀上学习，快速提升自己的 Kernel 技术实力，参与到 Linux Kernel 的开发中，成为顶级开源社区的贡献者！
