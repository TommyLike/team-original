**Keynote演讲**

这不是一场“讲概念”的分享，而是一次方向级解读 + 实践级展示：

- openEuler 超节点 OS：从管理硬件和物理资源到编排AI算力，易用性、可靠性、低时延能力进一步增强
- DevStation开发者工作站全新升级：软件包选型基于openEuler Master分支持续更新，提供更新、更易、更智能的AI开发体验
- openEuler将开启AgentOS时代：三层架构打造易用、安全、推理高效的Agent原生OS

如果你关心AI时代操作系统到底会变成什么样，这场一定不能错过！

**SIG Gathering专题研讨**

“我参与，我做主”——这次不只是听，是一起聊透。

不同于传统“听讲式”的会议，SIG Gathering 更像是一场真正属于开发者的“线下共创”。现场将设置 Kernel&基础服务 、AI生态支持&技术创新 、超节点技术创新 、具身智能&嵌入式 、AI时代高效开发与生态包管理 、全场景&Agent Infra 、RISC-V 、AI使能安全与漏洞处理 八大技术方向，汇聚不同 SIG 的维护者、贡献者、开发者与伙伴，共同展开更垂直、更深入的交流。

**AI Workshop**

如果你遇到过这些问题：

- GPU 被存储 I/O 拖慢
- PR 报错反复手修
- 故障定位全靠经验

那这个环节，建议你直接冲——全程实操 · 人手环境 · 现场跑通。

以下为五大高能玩法：

- 存储 I/O 极限挑战：亲手对比传统架构 vs AIDFS，引爆底层性能
- PR 自动修复 + 一键生成 PPT：从报错到修复再到汇报，Agent 一条龙全搞定
- AI 化开发全链路体验：DevStation + AI 助手，重塑开发流程
- “找茬”公测现场：已知问题分析 Agent，等你来“挑刺”
- 智能编程全家桶：Coding Agent + LLM 网关 + 性能看板，一套打通

**Poster Session** 

把你的技术成果，直接“秀”出来。

如果说演讲是“讲给大家听”，那 Poster Session 更像是面对面聊给你听。

技术成果不再停留在 PPT，而是以更直观的方式呈现在交流区。你可以近距离了解项目细节、实现路径，甚至在指定时间段与作者直接探讨技术细节。

让每一个成果，都有机会被看见、被讨论、被放大。

**开发者之夜**

白天聊不完的，晚上继续聊。当技术讨论从会场延续到夜晚，Developer Night 就成了最有“温度”的一站。

这里没有议程压力，也不设交流门槛。你可以继续白天的话题，也可以开启全新的对话；可以聊技术，也可以聊成长、聊社区、聊未来。

轻松的氛围、自然的交流，再加上一点点惊喜互动，让每一次对话都更有可能产生连接。

也许，一个新的合作、一段新的灵感，就在不经意间发生。
