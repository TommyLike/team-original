# [智能基座 openEuler 高校行——走进南京大学](https://mp.weixin.qq.com/s/wVItAidLboiAeVZuH93qQg)

[OpenAtom openEuler](javascript:void%280%29;)*2021-05-26 18:00:00*

2021 年 5 月 22 日，“Dev Run”智能基座 openEuler 高校行-南京大学专场在南京大学鼓楼校区软件学院成功举行。本次活动由华为计算产品线主办，南京大学、openEuler 社区、江苏鲲鹏生态创新中心协办，邀请多名专家来到现场，面向南京大学的师生讲解、分享基于 openEuler 开源社区的 Linux 学习模式、华为鲲鹏加速库技术及应用，并从职业发展的角度帮助南大学生深入认识鲲鹏计算产业及相关基础软件行业。

![](https://mmbiz.qpic.cn/mmbiz_jpg/A0h5yD51CMYF5wELIPyqlG3iaLBgS3mCWM1jTZibYodz0EH2NDb4NGOxCQUic7q1CyOVC2OTzmNT3ady34haNhRmA/640?wx_fmt=jpeg)

### 加速人才培养，应对数字化发展新阶段

随着 ICT 技术产业的迅猛发展，千行百业迎来了数字化转型的巨大机遇和挑战。为顺应新一阶段的发展要求、加速推进行业智能化建设，诸多行业都高度重视人才梯队的结构性调整，迫切需要引进大量高素质的数字人才。

南京大学软件学院副院长邵栋在开场致词中表示，南京大学和华为公司在产教融合协同育人领域的合作，集双方之所长，强强联合，为新工科领域的拔尖创新人才培养，开启了创新人才培养模式和方案。此次校园行活动是双方在“智能基座”协议签署后的又一实践活动落地，将帮助我校师生进一步深入了解计算产业、开源操作系统和鲲鹏领域相关技术等知识理论和实践应用。

![](https://mmbiz.qpic.cn/mmbiz_jpg/A0h5yD51CMYF5wELIPyqlG3iaLBgS3mCWQiaYRZTPvNYw5ERD6VN7FcnvSN77LZPlbicg326KWyLxW4qlEjTbNo7A/640?wx_fmt=jpeg)

江苏鲲鹏生态创新中心 COO 董亮表示：当前，行业数字化转型已经进入到智能升级的新阶段；智能产业与实体经济呈现紧密融合的趋势。然而，数字化转型越进阶，行业所面临的挑战就越严峻，进而对数字化产业生态提出更高的要求。智能基座项目以及 openEuler 高校行将持续深入，将为基础软件行业创新培养更多优秀人才，全力拓展行业发展。

![](https://mmbiz.qpic.cn/mmbiz_jpg/A0h5yD51CMYF5wELIPyqlG3iaLBgS3mCWNcSRibk4icEP6ks7uicKu9bxyC9uOocUBIZ2Uypz6AeI8VCf54Jmo37cg/640?wx_fmt=jpeg)

### 专家指点迷津，助学生深入了解计算产业

“教育部-华为‘智能基座’产教融合协同育人基地”项目于 2020 年 9 月启动，目前正在全国各地高校陆续落地。作为第一批共建“智能基座”产教融合协同育人基地的高校之一，南京大学已与华为就课程开发、课外实践活动、认证考试、实习就业等方面展开一系列合作。而本次南京大学专场作为 openEuler 高校行系列活动的第一站，也宣告华为就基础软件领域将切实与全国高校深入展开合作。

![](https://mmbiz.qpic.cn/mmbiz_jpg/A0h5yD51CMYF5wELIPyqlG3iaLBgS3mCWWXlN4WG2PLLYzkKqxyrou9dqxjcaSVkicltHbQLWicZLgTsicXepj4HrQ/640?wx_fmt=jpeg)

**基于 openEuler 开源社区的 Linux 学习新体验**

为了让广大学生更直观地感受 openEuler 开源社区，openEuler 社区布道师李宝林不仅趣味性地介绍了 Linux 的发展历史、开源文化、openEuler 社区及 openEuler 发行版的创新亮点，更引导学生基于 openEuler 开源社区、以多样的方式体验 Linux 操作系统的学习。

![](https://mmbiz.qpic.cn/mmbiz_jpg/A0h5yD51CMYF5wELIPyqlG3iaLBgS3mCWCcrVxCmduB0e0DTxsdMbXmTqjZ8oWdIcjhMWDDXibibWFG0K8yz1ImzA/640?wx_fmt=jpeg)

**华为鲲鹏加速库技术及应用**

从一道兔子繁殖的数学题切入，鲲鹏数学库架构师薛永辉深入浅出地向学生讲述华为鲲鹏加速库技术，启发学生单纯提升硬件计算能力，并不能更快解决问题，必须通过改变软件代码的数据结构和算法来提升基础软件库的计算效率。

![](https://mmbiz.qpic.cn/mmbiz_jpg/A0h5yD51CMYF5wELIPyqlG3iaLBgS3mCWHaIFL6HjFWMpjqrj0WQp7YIOalUIgzOl7Un0VbCKHF6icCSLEyBhmpA/640?wx_fmt=jpeg)

**平台成长-——职业发展讲座**

基于鲲鹏计算产业的蓬勃发展及行业数字化转型的广阔需求，华为计算产品线人力资源经理李春梅女士通过“点-线-面”法则，向学生介绍了如何选择发展方向及进行职业规划。

![](https://mmbiz.qpic.cn/mmbiz_jpg/A0h5yD51CMYF5wELIPyqlG3iaLBgS3mCWOzcG6ew3l1Fzibpvy7yicLdYWYzOdhLQ8uybh21YC8Ric76rlEHxQqTaQ/640?wx_fmt=jpeg)

创新研发，人才先行。智能基座 openEuler 高校行，是华为推进基础软件产学研用全面融合发展的重要举措，将与南京大学等诸多高校合力践行操作系统产业创新人才培养，让 openEuler 走进高校、走进课堂，让更多的学生学习和使用 openEuler，让 openEuler 成为最具活力的开源社区和协同创新平台，助推操作系统产业生态的繁荣与发展，为行业数字化转型和产业智能化升级提供长远动能。
