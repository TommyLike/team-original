# [【活动回顾】openEuler RISC-V Meetup北京站](https://mp.weixin.qq.com/s/NglPtjjlYpPWHFC0C3budw)

[OpenAtom openEuler](javascript:void%280%29;)*2024-05-07 17:31:03中国香港*

openEuler 24.03 新版本发布在即，为了促进开发者和社区用户之间的交流，2024年4月27日， OpenAtom openEuler（简称"openEuler"） RISC-V SIG 联合中科院软件所和开源之夏在北京举办了一场 Meetup 活动。此次 Meetup 汇聚了来自全国各地 40 余名专家朋友们，交流内容集中在 openEuler 在 RISC-V 架构的生态建设与计划和24.03LTS RISC-V 架构的发版亮点。同时 RISC-V SIG 还介绍了开源之夏 RISC-V 相关议题，以及如何参与 openEuler社区建设。

下面就让我们来回顾本次 Meetup 的内容

“

openEuler RISC-V 与开源之夏

![](https://mmbiz.qpic.cn/mmbiz_jpg/jqTxnpIZ2mA1qrcH0Z0icaDz3icuiaba0ZGZlgB311xaZCwvicrZhnMxriaxOCgD8NlIkxJq3Th0eYXNic3hpOMTPYQA/640?wx_fmt=jpeg&from=appmsg)

openEuler RISC-V SIG Maintainer 王经纬为我们详细介绍了openEuler  RISC-V SIG 的工作和进展，包括在 openEuler 上的软硬件适配工作成果、与厂商的合作情况、实习生培养计划等。除此之外，王经纬还为我们介绍了今年 RISC-V SIG 在开源之夏推出的 13 个项目。欢迎感兴趣的朋友们报名咨询。

“

LLVM 平行宇宙计划的进展报告

![](https://mmbiz.qpic.cn/mmbiz_jpg/jqTxnpIZ2mA1qrcH0Z0icaDz3icuiaba0ZGkCak7Y8gFF1XArNOEkWNrKpuXZ3xSdpdtBib8OCDHRU2RT58c2uvSlw/640?wx_fmt=jpeg&from=appmsg)

openEuler RISC-V SIG Maintainer 周嘉诚为大家解释了 LLVM 的背景及相关技术。在分享中，周嘉诚用图标等数据从性能以及场景热度的角度解释了 LLVM 的特点以及为什么想要换用 LLVM 来替代 GCC，并解答了台下嘉宾对 LLVM 场景的相关疑惑。

“

用 UEFI 启动一台 RISC-V机器

![](https://mmbiz.qpic.cn/mmbiz_jpg/jqTxnpIZ2mA1qrcH0Z0icaDz3icuiaba0ZGrZFfCZmHl8wEYKHVsicbzuLzrNH9RluTvwHDrfobXsgpu59mOk8qKzQ/640?wx_fmt=jpeg&from=appmsg)

openEuler RISC-V SIG Committer 汪流为大家介绍了在开发板用UEFI启动操作系统的各个阶段，以及各个阶段要达到的目的和实现目的所需要执行的代码等，如果你饱受软件虚拟 RISC-V 速度慢的困扰，不妨跟着汪流的操作启动一台硬件真实的 RISC-V。

“

OLK 6.6 同源计划

![](https://mmbiz.qpic.cn/mmbiz_jpg/jqTxnpIZ2mA1qrcH0Z0icaDz3icuiaba0ZGjWUoB5hJDrcv124RDkyKMr8xrcpibcZW75VcYnU8siaThTdrTiaLJh8sg/640?wx_fmt=jpeg&from=appmsg)

openEuler RISC-V SIG Committer 邢明政在分享中介绍了openEuler Kernel 和主线 Kernel 的区别。Kernel 作为一个持续演进的开源项目自然会产生很多补丁，在分享中，邢明政为大家强调介绍了 openEuler Kernel 补丁提交以及合入的规范。之后为大家介绍了RISC-V SoC 内核和内核现状所面临的一些现实问题。

“

如何建设基于 RISC-V 架构的 Uniproton

![](https://mmbiz.qpic.cn/mmbiz_jpg/jqTxnpIZ2mA1qrcH0Z0icaDz3icuiaba0ZGYibiazibyDh0jichfwiaaPreNZP1eAIV9mj89T5MKgiasqDme7EnGjb8PuAA/640?wx_fmt=jpeg&from=appmsg)

openEuler RISC-V SIG Committer 罗君介绍了 UniProton 这款实时操作系统的特点以及目前RISC-V 上对 UniProton 的支持情况。还为大家分享了如何建设基于RISC-V架构的 UniProton。

“

优化 openEuler RISC-V 的 JDK 环境

![](https://mmbiz.qpic.cn/mmbiz_jpg/jqTxnpIZ2mA1qrcH0Z0icaDz3icuiaba0ZGCLvlVblED6LarrOVTgiaYwB9QCMmFnE4KGibc9sEENOulZof6c6dkibHQ/640?wx_fmt=jpeg&from=appmsg)

中科院软件所工程师张定立为我们详细分析了 Zero JDK 在 RISC-V 下的性能问题，包括 TestTimedOut, OutOfMemoryError等，并提出了相应的解决版本。针对 JDK 的一些场景问题例如编译时版本混用、升级不规范也提出了实际的解决方案，张定立的分享为正在编译/修改 java 包的朋友们提供了帮助。

“

FFmpeg RISC-V 优化介绍

![](https://mmbiz.qpic.cn/mmbiz_jpg/jqTxnpIZ2mA1qrcH0Z0icaDz3icuiaba0ZGiciaKOOCvp97Y9Ju0FTicxcJ1wAqwGUqn7XoTdRvcOyjWZVicXTQUsUz9w/640?wx_fmt=jpeg&from=appmsg)

中科院软件所工程师孙越池介绍到FFmpeg 是一个开源的，重要的多媒体方面的基础库/工具，孙越池介绍了可优化的方向例如Vector编程模式、对齐方式、multipler后， 还用实际的代码案例，先找出代码中的问题，再根据问题提优化方案，为大家身临其境的介绍了FFmpeg 中的一系列优化手段。

“

在 RISC-V 国际基金会和 RISE 基金会的工作日常

![](https://mmbiz.qpic.cn/mmbiz_jpg/jqTxnpIZ2mA1qrcH0Z0icaDz3icuiaba0ZGkicytibOtU0JYIGRatsiaNxPV8icWuY043bXM5rnyUeB54Q6OAhGnic08Bg/640?wx_fmt=jpeg&from=appmsg)

openEuler RISC-V SIG Maintainer 吴伟详细介绍 RISC-V 国际基金会和 RISE 基金会的主要工作日常。吴伟老师还为大家打破了信息差和顾虑，为大家分享RISC-V 国际基金会和 RISE 基金会的加入标准，建议大家可以加入RISC-V 国际基金会和 RISE 基金会，还解答了服务区专家对于超算的相关疑惑。

![](https://mmbiz.qpic.cn/mmbiz_svg/TTQibyKjrickzz8RiaFM5H0Pae8WABgDLbSyW0becsEADibGIvznFy9pOodO3NhaEQhVF6libsFp0iaF2PmZIU2kFibvUwyesoWhW19/640?wx_fmt=svg&from=appmsg&wxfrom=5&wx_lazy=1&wx_co=1)
