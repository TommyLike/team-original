openEuler 社区作为暑期2021的联合主办方，此次上线了153个项目任务，涉及操作系统底层、云原生、虚拟化、安全、存储、桌面应用、开源基础设施建设等各个领域。这些任务由openEuler社区不同的SIG（Special Interest Group）出具，欢迎大家根据自身兴趣参与到各个SIG的项目中，来体验一把成为openEuler开源社区贡献者的乐趣吧！

 

![](https://mmbiz.qpic.cn/mmbiz_png/e7ibMUM1gDydnKXX5S4yUqCE2AW8tjjtQOGj8ckHG99IGLIPT2RMC95srEpChRicVkhjtx8YPSKQwrzf5T3veKkA/640?wx_fmt=png)

**内核方向**

openEuler kernel SIG 出具了40余个内核方向的项目任务，涵盖内核调度、内存、文件系统、架构、调测多个模块。

 

openEuler kernel 衍生自 Linux kernel，并在可靠性、安全性、虚拟化、性能等方面做了针对性的优化和增强，致力于打造最具活力的产业 Linux开源平台。

openEuler kernel SIG 聚集了国内外各领域技术专家和优秀的 Linux kernel 技术开发者，有浓厚的技术氛围，为大家提供丰富的学习资源和技术指导，涵盖体系结构、文件系统、内存管理等核心模块的基础及进阶知识， 深入浅出地带你探索 Linux kernel 。

 

在 openEuler kernel SIG，你将不断进步并发挥自己的价值，从一个开发者，逐步成为 kernel 社区的 committer、核心模块的 maintainer。你的技术能力将得到快速提升，成长为极具竞争力的 kernel 技术人才。你的代码将在国计民生、互联网等各行各业中创造价值，你将体验到技术改变世界的成就感！

云原生方向

openEuler CloudNative SIG出具了15+个云原生领域的题目，主要分为3类：开源集成项目、孵化项目和稳定特性的持续迭代项目。

开源集成项目相对来说比较简单，主要是完善云原生的生态，做一些云原生领域主流开源软件的集成，通过这些项目，可以熟悉和使用某个领域的开源软件，同时掌握操作系统的打包、编译、测试工程等知识；

孵化项目属于目前正在开发中的项目，本次只开放了eggo项目相关的任务。孵化项目需要有一定的开源软件开发经验，但是其中也有一个题目为体验类项目，从使用者的角度来发现孵化项目的问题，帮助改进项目的用户体验；

稳定特性的持续迭代这类项目，如容器引擎iSulad和容器镜像构建工具isula-build，在已经非常成熟的软件中，迭代开发一些小特性，通过这些项目让大家真正体验商用项目的代码提交过程，掌握参与开源贡献的方法，而且，每当想起自己写的代码正跑在全球各地的服务器上，是不是能收获些许小小的成就感呢？^\_^

虚拟化方向

 

Virt SIG致力于打造面向全场景、支撑多样性算力的虚拟化解决方案，提供高性能、高可靠和高安全的虚拟化组件。

Virt SIG在暑期2021里的项目任务，包括提升传统qemu-kvm虚拟化可维护性和用户体验的用户态热补丁技术“libcareplus”、虚拟机性能关键指标的工具“vmtop”特性开发、下一代虚拟化平台“StratoVirt”优化 和虚拟化测试套“EulerRobot”框架增强在内的13个趣味任务。

参与到Virt SIG暑期项目中，丰富的学习资源为你开放，为传统虚拟化的演进贡献自己的力量，在下一代虚拟化平台开发中留下自己的印记。在项目中收获的不仅仅是知识积累和技术实力提升，更重要的是在有趣的开发过程中，展现自己的影响力。在自己感兴趣的工作中作出成绩，幸福与快乐不就来了吗？看！Virt SIG下一个contributor头衔在朝你招手。

机密计算方向

机密计算是目前网络安全领域非常前沿的方向，是通过在基于硬件的可信执行环境中执行计算来保护使用中的数据的一种技术。

此次暑期2021活动中将有openEuler confidential-computing SIG顶尖的机密计算专家和兰州大学的老师带领大家一起探索机密计算技术，共同完善面向计算产业的机密计算安全应用开发套件SecGear。

项目任务方向包括：对操作系统部分基础软件的secGear化改造，用以提升系统重要执行流的安全性，例如对openssh的改造和graphene的引入，同时也包含许多基于secgear开发的创新性课题，在许多新兴领域发掘出机密计算的价值，例如基于secGear的数字作品权属保护等。

系统性能调优

A-Tune是一款基于openEuler开发的自动化、智能化性能调优引擎。它利用人工智能技术，对运行在操作系统上的业务建立精准模型，动态感知业务特征并推理出具体应用，根据业务负载情况动态调节并给出最佳的参数配置组合，从而使业务处于最佳运行状态。

 

A-Tune在暑期2021活动中有4个对openEuler系统中安装的具体应用进行调优的项目任务。

桌面

桌面相关的项目任务主要来自openEuler DDE SIG 和 UKUI SIG，出具了10+道丰富桌面应用相关的项目任务。

 

DDE是统信软件自研的一款全功能的 桌面环境。DDE作为一系列桌面组件的集合，不仅 为用户提供了符合统一规范且美观易用的图形化界 面，还为开发者提供了一整套界面美观且实用的 UI 图形库及开发套件（统信开发套件DTK）。

 

UKUI 是麒麟软件桌面研发团队开发的基于 Linux 发行版的轻量级桌面环境，其设计紧贴普通用户需求，特别是针对有一定 Windows 系统使用习惯的用户，减少其使用 Linux 系统的学习成本。全新的 UKUI 3.0 使用Qt开发，秉承 “友好易用，简单轻松” 的设计理念，将为用户提供更愉快的交互体验。
