# [「转」\[活动总结\] TARSIER 开放日暨 RISC-V 开源操作系统进展交流活动顺利举办](https://mp.weixin.qq.com/s/fUpv_EixUFNqiNV9Fi20wQ)

*张宇溪*[OpenAtom openEuler](javascript:void%280%29;)*2023-08-23 19:00:00*

2023年8月21日，由 TARSIER 团队组织的「TARSIER 开放日暨 RISC-V 开源操作系统进展交流」活动在中国科学院软件研究所顺利举行，该活动也是2023 RISC-V 中国峰会（8.23-8.25）的首批同期活动。

本次活动为RISC-V 开源操作系统开发人员搭建了一个集中技术交流平台，活动现场实物展示了RISC-V 桌面高性能计算、单板计算机、平板电脑等软硬件适配的最新成果，多位来自RISC-V 操作系统适配开发的一线工程师分享经验报告并参加圆桌讨论，为加强行业内学术交流、技术合作，推动RISC-V 开源操作系统的人才培养与创新提供目标与方向。**本次活动还分享了openEuler相关议题。**

开源操作系统对 RISC-V 的适配优化是完善 RISC-V 软件生态的重要抓手，TARSIER 操作系统团队自成立起始终坚持推进以及**完善包括 openEuler 在内的各大流行 Linux 发行版和硬件对 RISC-V 的支持。**
