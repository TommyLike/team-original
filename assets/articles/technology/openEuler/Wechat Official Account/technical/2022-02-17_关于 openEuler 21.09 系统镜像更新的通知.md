# [关于 openEuler 21.09 系统镜像更新的通知](https://mp.weixin.qq.com/s/syL4d3ZplgdysVflmmPLHw)

[OpenAtom openEuler](javascript:void%280%29;)*2022-02-17 22:09:26*

因默认repo地址发生变化，openEuler 21.09使用默认epol地址调用yum/dnf命令，将会发生错误。

为了解决这个问题，release SIG 决定将发布新的 openEuler 21.09 ISO 镜像，请使用 openEuler 21.09 的开发者尽快更新系统镜像，避免这种问题发生。

本次涉及更新的镜像如下：  
ISO

edge\_img

docker\_img  
stratovirt\_img  
virtual\_machine\_img

树莓派系统镜像的将在本通知发布30天之内更新。

如您不便更新系统镜像，可以使用以下方法更新 openEuler-repos 软件包，来解决这个问题：

```
dnf update openEuler-repos
```
