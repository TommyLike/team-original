# [openEuler 开源社区成立 OpenStack SIG，联通沃云成为该 SIG 的首位 Maintainer](https://mp.weixin.qq.com/s/vBBM798oM80Q70YrNwcW3Q)

*openEuler 团队*[OpenAtom openEuler](javascript:void%280%29;)*2020-07-08 21:03:17*

经 openEuler 技术委员会批准，openEuler 社区正式成立 OpenStack SIG，完善 openEuler 在云计算管理场景的能力支持，联通沃云作为当前 OpenStack SIG 的首位的 Maintainer，将承担主要适配工作。

![](https://mmbiz.qpic.cn/mmbiz_jpg/A0h5yD51CMaTeRYib9KXJDBZ6DDBeGOSgMiaU2o6rJzMX96JA9yFGNzcpLCVAbr1uLsdSgoF8QIVad9ibGS1dTw4Q/640?wx_fmt=jpeg)

openEuler 是一个在 2019 年开源的基于 Linux 稳定系统内核的服务器操作系统， openEuler 社区致力于通过打造开源社区，与全球的开发者共同构建一个开放、多元和架构包容的 openEuler 软件生态体系。

OpenStack 是以 Apache 许可证授权的开源云平台项目，最初于 2010 年由美国国家航空航天局和 Rackspace 合作研发，迄今为止已更新迭代了 22 个大版本，是目前市场上支持云环境最全面的开源云平台项目之一。

为了完善 openEuler 对 OpenStack 的适配，丰富 openEuler 在云平台领域的能力， openEuler 委员会批准成立了 OpenStack SIG，负责对 OpenStack 的专项适配工作。而作为 OpenStack SIG 首位 Maintainer 的联通沃云，将成为完全适配 OpenStack 工作的主要承担者。

对此，openEuler 技术委员会主席胡欣蔚表示：“***作为最重要的开源云平台软件之一，OpenStack 同时也是 openEuler 拓展数据中心软件生态的重要基础。联通沃云团队在 OpenStack 技术上有长时间的积累，也是 OpenStack 社区的活跃参与者。***

***这次主导在 openEuler 社区中围绕 OpenStack 成立 SIG，一方面是把 OpenStack带到 openEuler 上，为 OpenStack 拓展到更广阔的软硬件生态；另一方面也会帮助 openEuler 将数据中心生态的适应性提高到新的层次。openEuler 技术委员会非常欢迎沃云成为 openEuler 的重要贡献者，与  openEuler 社区中的虚拟化、容器团队密切合作，共同打造开源云解决方案的参考平台***“

联通云数据公司沃云计算组负责人陈硕表示：“***openEuler 操作系统的引入，对于联通沃云意义非常重大。在安可云计算领域，我们一直在尝试多种操作系统和多种应用平台架构的适配，openEuler 为我们提供了更加多元化的应用运行环境，同时也为鲲鹏强大计算能力的释放提供广阔前景。未来我们将致力于沃云基础平台在 openEuler 体系上更为稳固的适配，并为 openEuler 社区提供积极的反馈支持。希望通过参与 openEuler 社区，联通沃云能够进一步增强操作系统的自主适配能力，构筑沃云从操作系统到云平台的完整技术栈，丰富沃云硬件架构及自主可控能力。***“

作为中国联通倾力打造的云计算品牌，联通沃云是国内运营商中较早使用  OpenStack 开源框架的企业。早在 2010 年，联通沃云就开始积极探索使用  OpenStack 相关开源技术，彼时联通沃云的前身 —— 中国联通云计算研究团队，对市面上主流的开源云，包括底层虚拟化技术和云管理平台开源软件等进行了全面的技术考察，最终基于在开源实力、接口标准化、组件丰富度及耦合度等方面的优异性能，确定基于 OpenStack 加 KVM 打造初代沃云平台。

2013年，联通沃云成功基于 OpenStack 构建了沃云平台，并依托 OpenStack 不断更新迭代。2017年，中国联通入选 OpenStack 基金会黄金会员，标志着联通沃云基于 OpenStack 的云计算研发能力已达到国际先进水平。截至目前，沃云平台已经面向为超过百万家客户提供公共云、私有云和混合云服务，依托于中国联通丰富的网络资源及强大的运维服务体系，联通沃云已经成为政府、金融、互联网等领域客户的首选。

据了解，目前联通沃云已经结束了对鲲鹏计算平台的适配工作，并已完成 openEuler  内核适配方式的技术探索，OpenStack 全栈适配工作正在稳步进行中。

预计2020年年底，联通沃云将完成 OpenStack 平台组件在 openEuler 上的全栈适配，届时 openEuler 社区参与者和用户将可以通过社区获得基于 OpenStack 的完整云计算管理工具以及相关技术支持。

 

**关于 openEuler**

openEuler 是一个开源、免费的 Linux 发行版平台，将通过开放的社区形式与全球的开发者共同构建一个开放、多元和架构包容的软件生态体系。同时，openEuler 也是一个创新的平台，鼓励任何人在该平台上提出新想法、开拓新思路、实践新方案。

openEuler 通过社区合作，打造创新平台，构建支持多处理器架构、统一和开放的操作系统，推动软硬件应用生态繁荣发展。

 

**关于联通沃云**

2013年12月12日中国联通在北京举办了主题为“联通云世界 共享沃云时刻”的“沃云”品牌及产品发布会，正式发布了沃云品牌及沃云 V2.0 全系列云计算产品, 它是中国联通自主研发的面向企业和政府用户的云计算服务平台。沃云的底层平台基于 OpenStack 构建，目前已发展到了 6.0 版本，完成了 CMMI5、CSA-STAR、安全等保三级、ISO9001 、ISO 27018 等资质及多项可信云认证，并在电子政务、环保、医疗、教育、金融、旅游等各行业以及联通集团内部的私有云得到广泛应用。沃云以多层次的数据中心建设标准，已实现“4+31+N”资源布局，实现一省一池遍布全国。已拥有 590 个局站、近 2000 个机房，规模达到数百万核 CPU，EB级 存储，并仍在快速发展。
