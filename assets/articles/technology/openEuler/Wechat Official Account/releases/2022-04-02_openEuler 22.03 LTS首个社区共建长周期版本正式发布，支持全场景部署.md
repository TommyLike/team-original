2022 年 3 月 31 日，openEuler 22.03 LTS 如期而至。

openEuler 22.03 LTS 是欧拉完成正式捐赠后发布的首个共建社区版本。此版本也是欧拉首个支持全场景融合的社区长周期版本，满足服务器、云计算、边缘计算和嵌入式四大场景的多种不同类型设备部署要求和应用场景。作为长周期版本，openEuler 22.03 LTS 面向不同场景采用统一的 Linux Kernel 5.10 内核，面向服务器、云计算、边缘计算和嵌入式实现了统一构建、统一 SDK、统一联接，方便开发者构建面向全场景的数字基础设施操作系统。

openEuler 22.03 LTS 合入了 openEuler 三个创新版中经过商业验证的创新特性，并针对四大场景首次发布新增特性，共新增代码 2300 万行。

针对四大场景提供不同系统镜像：

- 服务器场景：openEuler 22.03 LTS Server
- 云计算场景：openEuler 22.03 LTS Cloud
- 边缘计算场景：openEuler 22.03 LTS Edge
- 嵌入式场景：openEuler 22.03 LTS Embedded

openEuler 22.03 LTS 全面支持多样性计算，包括鲲鹏、英特尔、飞腾、申威、龙芯、兆芯等主流芯片。全面支持 Intel Icelake，并且提供了对下一代 Intel Sapphire Rapids 的基础支持，其他特性将陆续合入到 openEuler 22.03 LTS 后续的更新中。

当前欧拉开源社区已正式发布适配硬件驱动、板卡硬件测试标准和测试自动化功能，支持基于型号查找，累计支持主流的芯片厂商有 Avago、Emulex、Intel、鲲鹏、龙芯、Mellanox、NVIDIA、飞腾、Qlogic、树莓派、RISC-V、兆芯、申威等，主流的整机厂商有百信、新华三、湘江鲲鹏、四川虹信、黄河科技、联想、宝德、曙光、清华同方、长江计算、超聚变、神州云科等，详情请查看欧拉开源社区官网硬件兼容性列表，欢迎更多芯片、板卡、整机厂商使用该功能完成硬件驱动适配。

**硬件兼容性列表**：https://www.openeuler.org/zh/compatibility/

在软件生态上，中科院软件所的 oepkgs 作为欧拉开源操作系统的第三方软件包扩展仓库，其中包括欧拉开源操作系统的 13000 个软件包，极大的丰富了欧拉开源操作系统的软件生态。

## 服务器场景

openEuler 22.03 LTS Server 首次发布高性能用户态协议栈 gazelle，MySQL、Redis、openGauss 等数据库性能相比内核态协议栈提升 5%。

新文件系统 EulerFS 正式发布到 openEuler 22.03 LTS 中，面向非易失性内存的新文件系统，采用软更新、目录双视图等技术减少文件元数据同步时间，提升文件读写性能。

内存分级扩展 etMem 正式发布到 openEuler 22.03 LTS 中，新增用户态 swap 功能，策略配置淘汰的冷内存交换到用户态存储，用户无感知，性能优于内核态 swap。

## 云场景

openEuler 22.03 LTS Cloud 支持在离线环境下的混合部署，支持在混合部署的情况下，在线容器对离线容器 CPU 的抢占，支持在内存不足时有限回收离线内存，在典型应用场景下，例如 MySQL 和 Redis 的在线和离线混合部署的场景下，业务响应时延抖动小于 5%，资源利用率提升 30%。

openEuler 22.03 LTS Cloud 支持 Libcareplus 虚拟化热补丁功能，支持 QEMU 在遇到紧急问题和安全漏洞的情况下进行热修复，批量修复补丁数量比开源基线版本大幅提升，支持一键式补丁制作，补丁制作效率提速 8 倍，支持补丁全生命周期管理和解析工具

虚拟化运行时 StratoVirt 安全容器支持直通设备热插拔。

KubeOS 正式发布到 openEuler 22.03 LTS 中，具备运行内存消耗&lt;150M，重启时间&lt;15 秒。

容器管理引擎 iSulad 正式发布到 openEuler 22.03 LTS 中，StratoVirt + iSulad 形成的容器解决方案相比于 Docker+QEMU，底噪和启动时间优化 40%。

双平面部署工具 eggo 正式发布到 openEuler 22.03 LTS 中，提供高效稳定的集群部署集群的能力，支持单集群多架构、支持在线和离线部署模式多种部署模式，结合 GitOps 管理能力、感知集群配置变化，驱动集群 OS 统一高效部署。

## 边缘场景

openEuler 22.03 LTS Edge 是针对边缘场景构建的系统镜像，具备易用、高性能、云边协同的特性， KubeEdge 边云协同框架正式发布到 openEuler 22.03 LTS Edge 中，支持边云网络 EdgeMesh 增强，时延降低 30%，降低 5%的丢包率。

## 嵌入式场景

openEuler 22.03 LTS Embedded 基于 Linux Kernel 5.10 内核提供软实时能力。集成鸿蒙分布式软总线能力，实现欧拉嵌入式设备之间的互联互通。实现 SoC 内实时和非实时多平面混合部署，支持 Zephyr 实时操作系统，支持 Zephyr 的消息发送与接收基本通信功能。支持基于 Yocto 构建包的小型化定制裁剪能力，自由配置基础文件系统，当前已经支持 80+ 嵌入式场景核心软件包。

## 统一构建&SDK

openEuler 22.03 LTS 首次发布 EulerMaker。EulerMaker 支持文件级别的裁剪，支持根据用户需求制作最小 100M 的系统镜像。发布机密计算 SDK，支持开发者快速构建机密计算应用。
