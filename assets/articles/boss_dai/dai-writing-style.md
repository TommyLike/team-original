# 戴老板笔法指南（饭统 / 饭桶戴老板 · 远川研究所）

> 本指南提炼自「饭统戴老板」公众号的原创长文，用于在生成**有故事性的 PDF/长文报告**时
> 借鉴其叙事笔法。**只借语感与结构，不借事实**——所有数据、事实以本项目的研究为准。
>
> 使用方式见文末「如何应用」。核心原则一句话：**用硬核的事实，讲有温度的故事。**

---

## 一、整体调性

- **定位**：有趣且深度的硬核财经。既有扎实的数据与考据，又有人文温度和历史纵深。
- **气质**：博学、戏谑、克制的幽默。像一个读书很多、见识很广、又爱开玩笑的朋友在跟你聊。
- **底色**：几乎每篇最终都会落到更大的命题上——国运、时代、周期、人性、产业兴衰。
  小切口进，大格局出。

---

## 二、结构：起承转合

戴老板的长文几乎都遵循一条稳定的骨架：

1. **起（钩子）**：用一个**具体的场景**开篇，而不是从概念讲起。
   - 常见三种钩子：①一条当下的新闻/现象（"2019年6月30日晚上8点，柳岩在快手做了一场直播"）；
     ②一段久远的历史（"1629年，大明王朝摇摇欲坠、国祚将尽"）；③一个人物的具体时刻。
   - 开篇往往**看似离题**（闲笔起兴），几段之后才显露与主题的关系——这种"绕一下"正是味道所在。
   - 钩子里塞满**精确的时间、地点、人名、数字**，让画面立起来。

2. **承（设问）**：铺垫几段后，抛出一个**贯穿全文的核心问题**，作为全文引擎。
   - 典型句式："我们的问题随之而来：华语乐坛究竟是如何走向'沉寂'的？"
   - 这个设问要在结尾被回答或升华。

3. **转（主体）**：用**编号小节**（`01 / 02 / 03` 或 `1. / 2. / 3.`）拆解。
   - 常把论点包装成一套**框架或"原理"**（如《网红经济学》拆成"四条经济学原理"）。
   - 每个小节**独立起一句凝练的钩子句**，单独成段，像小标题下的第一记重拳：
     "刘备的发迹史，就是一部网红大V炼成史。"
   - 小节可加**题记**：一句歌词、名人语录或脚注（如以李宗盛《山丘》歌词起小节）。

4. **合（收尾）**：**回扣开头**，并升华到时代/国运/人性层面，留一句余味悠长的话。

## 三、论证手法

- **历史类比**：用一个远隔时空的故事映射当下。
  - 例：刘备从"卖草鞋"（高边际成本）到"卖仁义人设"（零边际成本、可无限复制）＝ 网红炼成。
  - 例：1629 年裁驿站逼反李自成 → 明亡；300 年后 1911 年"铁路国有"→ 辛亥革命。两次"物流变故动摇国运"并置。
- **中外/古今并置对照**：把两条线索并排放，让反差自己说话。
- **数据支撑 + 出处**：关键论断配具体数字，重要引用带脚注 `\[6]`。数字要"感人"地具体（"直播2分钟卖出2.5万单"）。
- **由小见大**：从一个带货主播/一匹驿马/一首歌，推到边际成本、定位理论、时代情绪。

## 四、语言风格

- **雅俗混搭**：文言、成语、典故 ⇄ 网络热词（种草、giegie、割割、Z世代）。一句庄、一句谐，落差制造幽默。
- **化用经典句式**（点到为止，不滥用）：
  - 鲁迅腔："办公室内外充满了快活的空气""众人都哄笑起来"。
  - 木心/民谣腔："通信基本靠吼，车马很慢……"。
  - 毛泽东《湖南农民运动考察报告》腔："到抖音去，到快手去，到流量中心去。"
  - 张爱玲："出名要趁早"。
- **金句收束**：段末常有一句对仗/排比/警句压轴。"中文越来越新，中文歌却越听越老。"
- **排比与递进**："归置房间、室内装修、宠物培训……任何一技之长在边际成本归零后，就足以实现财富自由。"
- **自嘲式幽默**（克制）："两百多斤的楚团长，本想学怎样当网红，结果看了10分钟就入手了两支精华水。"
- **节奏**：段落短，一段一个意思；长短句交替；叙述推进快，不拖泥带水。

---

## 五、明确要避免的

- ❌ **堆砌数据不讲人**：数字要服务于故事和人物，不是罗列。
- ❌ **幽默过火/油腻**：梗要克制，一段一两个即可，点到为止。
- ❌ **说教与口号**：观点藏在事实和故事里，让读者自己得出，而非直接喊结论。
- ❌ **为文采牺牲准确**：化用句式和金句不能扭曲事实与数字。
- ❌ **平铺直叙从概念开篇**："本文将分析……"这种开头是戴老板笔法的反面。

## 六、一页速查（写作时对照）

| 环节 | 该做什么 |
|------|----------|
| 开头 | 一个具体场景/新闻/历史时刻切入，塞满时间地点人名数字，允许"闲笔绕一下" |
| 第 3-5 段 | 抛出一个贯穿全文的核心设问 |
| 主体 | 编号小节；每节一句独立的钩子句；把论点包装成框架/"原理" |
| 论证 | 历史类比 + 古今中外并置 + 带出处的数据 |
| 语言 | 雅俗混搭、克制化用经典句式、段末金句 |
| 结尾 | 回扣开头，升华到时代/国运/人性，留一句余味 |
| 全程 | 有趣且深度、有温度、克制幽默；事实数字必须准确 |

## 七、如何应用（生成报告时）

1. **先读本指南**，建立对笔法的整体认知。
2. **再通读 5 篇原文**（路径见 `STORYTELLING-REFERENCE.md`），优先挑与本报告**题材相近**的篇目
   （商业/科技/历史/地缘/人物），感受真实的语感、节奏、钩子与收尾。
3. **先整理、后动笔**：读完后，先针对**本报告主题**写一段"笔法应用要点"草稿（开头怎么起、
   核心设问是什么、用哪个历史类比、结尾升华到什么），再开始写正文叙事段落。
4. **只用于叙事段落**：数据表、方法论、结论清单等仍用中性、精确的表达。
5. **事实优先**：语感是借来的，事实和数字永远以本项目的研究为准。
